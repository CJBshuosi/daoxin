# 倒推分析：对标花火Spark，提升文案输出质量

> 基于花火Spark的实际生成流程对比，倒推道心文案需要研究和改进的内容。

## 一、花火输出了什么（目标）

```
选题标题 + 执行思路（镜头语言 + 剪辑节奏 + 情绪曲线 + 结尾引导）+ 完整文案
```

## 二、要输出这些，AI 需要知道什么

| 所需知识 | 花火怎么做的 | 我们现在缺什么 |
|---|---|---|
| 用户画像 | 对话式收集（行业/产品/目标用户） | 只有赛道名+描述，太粗 |
| 内容策略体系 | 守正/出奇 → 16+子方向 | 3种框架，太浅 |
| 选题方法论 | 每个子方向有独立的选题逻辑 | 只做四维度+欲望分析 |
| 短视频制作知识 | 镜头语言、剪辑节奏、BGM节奏 | 完全没有 |
| 情绪设计知识 | 情绪曲线、节奏起伏设计 | prompt 里提了但很浅 |
| 爆款案例库 | 背后有4年教学数据 | 只有 fewShot 一个文本框 |

## 三、需要研究的内容（按优先级）

### P0 — 必须研究，直接影响输出质量

#### 1. 短视频爆款方法论
- **研究对象**：亲爱的安先生、秋叶大叔、卡思数据的短视频方法论
- **重点**：选题角度分类体系、爆款公式、内容结构模型
- **目的**：替代我们现在过于简单的3框架体系

#### 2. 短视频镜头语言与剪辑节奏
- **研究对象**：抖音创作者学院、剪映官方教程
- **重点**：口播类/图文类/混剪类的镜头模板、节奏模板
- **目的**：让 AI 能输出具体的拍摄执行方案

#### 3. 情绪曲线设计
- **研究对象**：故事弧线理论（Dan Harmon Story Circle）、短视频完播率研究
- **重点**：3秒/7秒/15秒/30秒/60秒 不同时长的情绪节奏模板
- **目的**：文案输出附带情绪曲线标注

### P1 — 重要，影响生成精准度

#### 4. 用户画像与内容定位方法论
- **需要收集**：行业背景、变现产品、目标用户年龄/痛点/场景、账号人设
- **目的**：生成前有足够上下文，不再是"给个主题就生成"

#### 5. 垂直赛道的爆款规律研究
- 国学赛道：什么选题角度完播率高？什么表达方式点赞高？
- 养生赛道：合规红线在哪？什么钩子最有效？
- **目的**：每个赛道有独立的 prompt 知识库

### P2 — 锦上添花，提升体验

#### 6. 平台算法逻辑
- 抖音/小红书/视频号的推荐机制差异
- 不同平台的内容偏好和违禁规则
- **目的**：按平台优化输出

## 四、需要改进的具体模块

### 流程改进

```
改进前（当前流程）：
  主题输入 → 选题分析 → 选框架(3种) → 选钩子 → 生成文案 → 润色

改进后（目标流程）：
  用户画像收集 → 策略选择(守正/出奇+子方向) → 选题生成(3个+执行思路)
  → 选题确认 → 文案大纲(含情绪曲线) → 完整文案+拍摄指导 → 润色
```

### 模块改进清单

| 模块 | 改什么 | 怎么改 |
|---|---|---|
| **赛道配置** | 增加用户画像字段 | Track 加 targetAudience、product、persona 字段 |
| **Step 0（新增）** | 用户信息收集 | 首次使用赛道时引导填写画像 |
| **Step 1** | 从"分析选题"变为"生成选题+执行思路" | 重写 prompt，加入方法论知识 |
| **Step 2** | 从3框架变为策略体系 | 守正(9方向)/出奇(8方向)，每个有独立 prompt |
| **Step 3** | 钩子生成不变，但要更丰富 | 加入镜头语言建议 |
| **Step 4** | 文案输出格式升级 | 加情绪曲线标注、镜头提示、剪辑节奏建议 |
| **Prompt 知识库** | 核心改进点 | 把方法论写成结构化 prompt 模板 |

## 五、最关键的一步

所有改进的核心是 **prompt 知识库的深度**。花火的优势不是代码，是背后4年积累的方法论被结构化注入了 prompt。

### 建议行动路径

1. **先研究方法论**（1-2周）— 整理出完整的选题策略体系和短视频制作知识
2. **写成结构化文档** — 保存到项目里作为 prompt 知识库
3. **重写 prompt 模板** — 基于知识库重构所有 step 的 prompt
4. **调整数据结构和流程** — 扩展 Track 类型、增加步骤
5. **测试对比** — 用同样的主题对比输出质量

## 六、对比案例

### 测试主题：「最积德的三件事，做过一件就是大福报」（国学赛道）

**花火Spark 输出（选题阶段）**：
- 选题1：《为什么我们读了那么多国学经典，却依然过不好这一生？》+ 完整执行思路（镜头、剪辑、情绪）
- 选题2：《<道德经>里藏着的职场生存法则，你真的看懂了吗？》+ 完整执行思路
- 选题3：《如果孔子活在今天，他会怎么发朋友圈？》+ 完整执行思路

**道心文案 输出（选题阶段）**：
- 2-3句话的选题分析
- 诉求维度评估
- 底层欲望判断
- 一句话优化建议

**差距**：花火在选题阶段就提供了可执行的创意方案，我们只做了分析评估。
