## 二、爆款选题模式（Viral Topic Patterns）

### 2.1 高频爆款主题类型

| 主题类型 | 典型选题 | 爆款原因 |
|---------|---------|---------|
| 古人智慧+现代应用 | "王阳明心学教你职场破局"、"鬼谷子识人术，三招看透同事" | 实用性强，直接解决当下痛点 |
| 经典新解 | "《道德经》里藏着的赚钱逻辑"、"《孙子兵法》与创业的底层逻辑" | 反差感，古今碰撞引发好奇 |
| 历史人物故事 | "苏东坡被贬三次，为什么越活越通透"、"曾国藩从笨人到圣人的逆袭" | 人物弧光+励志共鸣 |
| 古今对比 | "古人的社交规则vs现代职场潜规则"、"《菜根谭》里的断舍离智慧" | 降低理解门槛，增强代入感 |
| 处世智慧 | "老祖宗说：腊月不做三件事"、"为什么说'穷不走亲，富不还乡'" | 民间智慧+生活化，传播性强 |
| 诗词意境 | "李白最孤独的一首诗，读懂已是中年人"、"辛弃疾最霸气的十个字" | 情感共鸣+文化自豪感 |
| 禁忌/冷知识 | "古人为什么说'男不拜月，女不祭灶'"、"筷子为什么是7寸6分" | 好奇心驱动，社交货币 |

### 2.2 选题叠加公式

```
爆款选题 = 经典IP × 现代痛点 × 情绪触发
```

- **经典IP**：《道德经》《论语》《易经》《菜根谭》《孙子兵法》《鬼谷子》《曾国藩家书》
- **现代痛点**：职场内卷、人际关系、焦虑迷茫、婚姻家庭、子女教育、财富认知
- **情绪触发**：恍然大悟感、文化自豪感、醍醐灌顶感、怀旧感、敬畏感

### 2.3 周期性爆款节奏

- **节气节点**：每个节气前1-2天发布对应节气文化内容
- **传统节日**：春节（家训/年俗）、清明（孝道）、端午（屈原/爱国）、中秋（思乡/团圆）、重阳（敬老/登高）
- **开学季**：9月前后，教育类国学选题爆发
- **年终**：12月-1月，总结类/新年寄语类国学内容

## 四、高效钩子模板（Effective Hooks）

### 4.1 反常识型

```
"你以为{经典观点}？其实{颠覆性新解}。"
```
- "你以为《道德经》是消极避世？其实老子比任何人都懂竞争。"
- "你以为孔子温文尔雅？他其实是春秋时期最硬核的人。"
- "都说'吃亏是福'，老祖宗的原话可不是这个意思。"

### 4.2 权威数字型

```
"{数字}年前，{人物}说了{N}个字，如今字字应验。"
```
- "2500年前，老子写下81章五千言，其中第33章，读懂的人都活明白了。"
- "曾国藩一生写了1500封家书，反复提到的只有两个字。"
- "鬼谷子只收了四个徒弟，却搅动了整个战国。"

### 4.3 情感共鸣型

```
"如果你最近{负面状态}，请一定看完这条。"
```
- "如果你最近觉得特别累，又不知道为什么，苏东坡的这句话你一定要听。"
- "如果你正在经历人生低谷，请记住王阳明在龙场悟到的那句话。"
- "三十岁以后才读懂的一句话，以前觉得是鸡汤，现在字字扎心。"

### 4.4 故事传奇型

```
"{人物}在{极端处境}下，说了一句话，流传了{N}年。"
```
- "苏东坡被贬到最远的海南岛，写下了一生最豁达的十个字。"
- "王阳明在贵州龙场，九死一生之后，悟出了四个字。"
- "曾国藩打了人生最惨的败仗之后，给自己写了六个字。"

### 4.5 植入欲望型

```
"真正{高段位形容}的人，都懂{具体智慧}。"
```
- "真正厉害的人，从不解释自己。老子两千年前就说透了。"
- "层次越高的人，越不争辩。这就是《道德经》讲的上善若水。"
- "格局大的人都有一个共同点，《论语》里一句话说透了。"

### 4.6 好奇悬念型

```
"古人说'{俗语/名句}'，但后半句才是精华，99%的人不知道。"
```
- "'人不为己，天诛地灭'——这句话被误解了两千年。"
- "'天行健，君子以自强不息'的下一句，才是真正的大智慧。"
- "老祖宗说：'穷不走三亲，富不交三友'，指的到底是哪三种？"

## 五、有效内容角度

### 5.1 古今对比角度

**公式**：古代场景 + 现代翻译 + 实际应用

```
选题示例："孔子的'因材施教'，就是现代管理学的OKR"
结构：古人原话 → 白话解读 → 当代职场/生活场景映射 → 金句收尾
```

**优势**：降低理解门槛，让观众觉得"国学不遥远"

### 5.2 经典新解角度

**公式**：经典名句 + 被误解的真相 + 正确解读带来的顿悟

```
选题示例："'无为而治'不是什么都不做，而是不做多余的事"
结构：常见误解 → 原文语境还原 → 真实含义 → 对当代的启示
```

**优势**：反常识带来的"恍然大悟"感，完播率极高

### 5.3 人物故事角度

**公式**：极端处境 + 人物选择 + 背后的哲学 + 现代启示

```
选题示例："苏东坡三次被贬，为什么越贬越通透"
结构：人生转折点 → 困境中的选择 → 背后的思想 → 对你我的意义
```

**优势**：故事性强，情绪弧光明显，易引发共鸣

### 5.4 实用智慧角度

**公式**：具体生活场景 + 古人解法 + 可操作的建议

```
选题示例："遇到小人怎么办？鬼谷子教你三招"
结构：生活痛点 → 古人的做法 → 拆解为具体步骤 → 总结金句
```

**优势**：实用性强，收藏率和分享率高

## 六、常见错误与避坑

### 6.1 内容层面

| 错误 | 具体表现 | 正确做法 |
|------|---------|---------|
| 掉书袋 | 大段文言文引用，观众听不懂 | 白话为主，文言为点缀，每次最多引一句 |
| 伪国学 | 把鸡汤包装成国学，张冠李戴 | 引用必须标注出处，确保准确 |
| 说教感 | "你应该这样做"的居高临下语气 | 用"我最近才明白"的分享语气 |
| 内容空洞 | 只有感慨没有干货 | 每条必须有一个明确的信息增量 |
| 过度神化 | "古人什么都预测到了" | 客观看待，承认时代局限性 |
| 标题党 | 标题夸张但内容撑不起 | 内容价值要配得上钩子的承诺 |

### 6.2 制作层面

- **BGM太吵**：国学内容适合安静/古风BGM，不要用激烈的音乐
- **画面杂乱**：水墨风、书法字幕、简洁古风最佳
- **语速不当**：国学内容适合中等偏慢的语速，留有思考空间
- **视频太长**：新账号建议30-60秒，成熟账号1-3分钟

### 7.1 头部账号特征

| 风格类型 | 代表方向 | 内容特点 |
|---------|---------|---------|
| 大师讲座剪辑型 | 曾仕强、南怀瑾语录剪辑 | 权威感强，金句密度高，适合做语录号 |
| 知识讲解型 | 国学解读、经典新解 | 深入浅出，白话解经典，教育属性强 |
| 故事演绎型 | 历史人物故事、典故演绎 | 叙事性强，情绪饱满，完播率高 |
| 生活智慧型 | 国学+生活场景 | 接地气，实用性强，分享率高 |
| AI国学型 | AI数字人+水墨动画 | 制作门槛低，视觉新颖，适合矩阵 |

### 7.2 成功账号共性

1. **人设清晰**：或儒雅书生，或通透长者，或文艺青年
2. **视觉统一**：封面风格一致（水墨/书法/古风），辨识度高
3. **金句密度高**：每15秒至少一个可截图的金句
4. **评论区运营**：置顶评论引导互动，回复粉丝问题
5. **系列化内容**：如"道德经逐章解读"、"古人识人术系列"

### 7.3 标准爆款视频结构

```
30秒视频 = 5秒钩子 + 15秒核心内容 + 5秒金句升华 + 5秒互动引导
60秒视频 = 5秒钩子 + 40秒故事/论证 + 10秒金句升华 + 5秒互动引导
```

## 八、合规红线与注意事项

### 8.1 政治敏感红线

- **禁止**歪曲中华优秀传统文化，不得恶搞经典人物
- **禁止**利用国学宣扬封建迷信（算命、风水等明确的迷信内容）
- **禁止**损害革命领袖、英雄烈士形象
- **禁止**涉及民族分裂、地域歧视的内容
- **谨慎**使用涉及历史评价有争议的人物和事件

### 8.2 内容准确性要求

- 引用经典原文必须准确，不得篡改原意
- 人物故事不得虚构历史事实（可以合理推演，但要标注"演绎"）
- 不得将不同时代、不同学派的思想混为一谈
- 翻译/解读有争议时，应注明"一种解读"而非"这就是原意"

### 8.3 商业合规

- 不得利用国学概念进行虚假宣传（如"学了这个就能成功"）
- 知识付费产品不得夸大效果
- 引流到私域不得涉及违法违规服务（如算命收费等）

### 8.4 平台规则

- **AI内容标注**：使用AI生成的内容建议主动标注
- **版权注意**：使用古风音乐、书法作品等注意版权
- **绝对化用语**：避免"最"、"第一"、"唯一"等绝对化表述
- **评论区引流**：不得在评论区放明确的联系方式/链接

### 8.5 安全表达替换表

| 敏感表达 | 安全替代 |
|---------|---------|
| "这是唯一正确的解读" | "这是一种很有启发的解读" |
| "古人早就预测到了" | "古人的智慧在今天依然有参考价值" |
| "学了就能改变命运" | "这种思维方式或许能给你新的启发" |
| "风水/命理说..." | 避免，改为"古人的处世哲学" |
| "某某朝代最伟大的..." | "某某朝代非常有影响力的..." |

## 九、AI Prompt 注入要点

当生成国学赛道文案时，系统应自动注入以下约束：

```
【国学赛道知识注入】
1. 引用经典时标注出处（书名+章节）
2. 以白话为主，文言点缀，确保口语化表达
3. 每条内容需有明确的"信息增量"（新知、新解、新视角）
4. 触发情绪：顿悟感、文化自豪感、醍醐灌顶感
5. 钩子类型优先：反常识 > 权威数字 > 故事传奇
6. 金句密度：每15秒一个可记忆的短句
7. 禁止：封建迷信、篡改经典、说教语气、绝对化表述
8. 结尾引导：用开放性问题引发评论讨论
```
