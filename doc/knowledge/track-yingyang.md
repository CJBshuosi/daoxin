## 三、爆款选题类型（映射道心四法）

### 3.1 选题类型矩阵

| 选题类型 | 典型选题 | 道心四法 | 爆款原因 |
|---------|---------|---------|---------|
| 认知颠覆型 | "你以为很健康的5种食物，其实热量炸裂"、"全麦面包可能比白面包还不健康" | 明道·洞见 | 打破健康饮食误区，认知落差感强 |
| 真相揭秘型 | "超市里的'0糖'饮料真的不含糖吗？看完配料表你就懂了"、"酸奶的营养骗局" | 明道·洞见 | 揭秘感强，高分享率 |
| 减脂记录型 | "靠吃减了20斤，分享我的30天三餐记录"、"从140到120，不节食的减脂饮食全分享" | 动心·共鸣 | 真实感强，激励效果显著 |
| 健康转变型 | "控糖一个月，身体发生了这些变化"、"戒掉外卖30天的真实记录" | 动心·共鸣 | 生活化记录引发认同和效仿 |
| 食谱教程型 | "一周减脂餐不重样，每顿不超过15分钟"、"5款低卡高蛋白早餐，上班族10分钟搞定" | 启思·价值 | 拿来即用，收藏率极高 |
| 食材科普型 | "10种高蛋白食物排行榜"、"升糖指数最低的主食TOP10" | 启思·价值 | 工具型内容，长尾流量好 |
| 反常识观点型 | "减脂不需要戒碳水，正确吃碳水反而瘦得更快"、"吃肥肉不一定长胖，关键看这个" | 破局·创意 | 挑战传统认知，引发争论和讨论 |
| 创意吃法型 | "用电饭煲做出餐厅级减脂餐""把西蓝花做到比炸鸡还好吃的方法" | 破局·创意 | 新奇实用，降低健康饮食门槛 |

### 3.2 选题公式

```
爆款选题 = 饮食目标/健康痛点 × 食材/食谱 × 简单可执行
```

- **饮食目标**：减脂、控糖、降尿酸、增肌、养胃、抗炎、通便
- **食材/食谱**：早餐、午餐、晚餐、加餐、饮品、零食替代
- **简单可执行**：快手（10分钟）、懒人、外卖族、一人食、便当

### 3.3 周期性爆款节奏

| 时间节点 | 爆款方向 |
|---------|---------|
| 1-2月（春节前后） | 年后减脂餐、解油腻食谱、春节饮食控制 |
| 3-5月（减脂季） | 夏天前减脂冲刺、露肉季饮食计划 |
| 6-8月（夏季） | 消暑饮品、夏日低卡食谱、水果热量科普 |
| 9-10月（秋季养生） | 秋季润燥饮食、换季养生汤、贴秋膘健康版 |

## 四、钩子模板

### 4.1 误区颠覆型（明道·洞见）

```
"你以为{很健康/很不健康}的{食物}，其实{相反的真相}。"
```
- "你以为很健康的燕麦奶，其实含糖量比可乐还高。别再被包装骗了。"
- "减脂期不敢吃米饭？其实白米饭的热量比你想象中低得多，关键在吃法。"
- "你天天喝的那杯'零糖'酸奶，翻过来看配料表，第二行就是代糖。"

### 4.2 食谱推荐型（启思·价值）

```
"{场景/目标}就吃这{N}道菜，{简单程度}，{效果承诺}。"
```
- "减脂期晚餐就吃这3道菜，热量不到400大卡，好吃到不像减脂餐。"
- "上班族5分钟早餐，高蛋白低GI，吃完一上午都不饿。"
- "控糖一周食谱，照着做就行，食材超市全能买到。"

### 4.3 食材揭秘型（明道·洞见）

```
"经常吃{食物}的注意了，{你不知道的影响}。"
```
- "经常喝粥的人注意了，白粥的升糖速度比白糖还快。"
- "每天吃水果的人注意了，这3种水果的热量堪比主食。"
- "天天吃外卖的人看看这个，一顿外卖的钠含量可能超标3倍。"

### 4.4 效果见证型（动心·共鸣）

```
"坚持{饮食调整}{时间}，{可见的变化}。"
```
- "坚持抗炎饮食一个月，皮肤状态肉眼可见地变好了。"
- "戒糖21天，体重没怎么掉，但脸上的痘痘几乎全消了。"
- "吃够蛋白质之后，减脂终于不掉肌肉了，体型变化明显。"

### 4.5 痛点直击型（动心·共鸣）

```
"如果你{饮食困扰}，问题大概率出在{根本原因}。"
```
- "如果你少吃了体重还不掉，问题不在吃多少，在吃什么。"
- "如果你总是下午三四点特别想吃甜的，说明你午餐的搭配有问题。"
- "如果你每次减脂都反弹，不是你意志力差，是你的代谢被节食搞坏了。"

### 4.6 对比清单型（启思·价值）

```
"{N}种{食物类型}热量/营养对比，看完再也不会选错。"
```
- "10种常见主食热量对比，减脂期选这3种就够了。"
- "6种常见奶制品营养对比图，别再喝错了。"
- "外卖常点的8道菜热量排行，第1名的热量是最后一名的5倍。"

## 五、内容角度

### 5.1 食谱教程角度（高收藏，核心内容）

**公式**：饮食目标 + 食谱展示 + 营养分析 + 操作步骤

```
选题示例："减脂期一周早餐不重样，每款10分钟搞定"
结构：目标说明（低卡高蛋白）→ 7款早餐展示 → 每款热量标注 → 简易做法
```

### 5.2 科普辟谣角度（高传播+建立专业度）

**公式**：常见误区/谣言 + 科学解释 + 正确做法 + 信息来源

```
选题示例："'食物相克'是真的吗？营养学怎么说"
结构：列出常见说法 → 科学解释为什么不成立 → 真正需要注意的 → 引用权威来源
```

### 5.3 配料表解读角度（高互动+涨粉）

**公式**：常见食品 + 配料表展示 + 成分解读 + 健康替代

```
选题示例："超市酸奶配料表大揭秘：哪些是真酸奶，哪些是'糖水'"
结构：拿多款产品 → 逐一解读配料表 → 教会看关键指标 → 推荐选择标准
```

### 5.4 饮食挑战/记录角度（高点赞+分享）

**公式**：饮食调整方案 + 执行记录 + 身体变化 + 心得总结

```
选题示例："戒掉添加糖30天，记录身体的真实变化"
结构：挑战规则说明 → 每周记录（饮食+感受）→ 体测数据变化 → 经验总结
```

## 六、常见错误

### 6.1 内容层面

| 错误 | 具体表现 | 正确做法 |
|------|---------|---------|
| 伪科学概念 | 使用"排毒""刮油""负卡路里食物"等伪概念 | 基于营养学和循证医学，用科学术语 |
| 极端饮食 | 推荐断食、极低热量饮食、单一食物饮食 | 推荐均衡饮食，强调个体差异和循序渐进 |
| 虚假效果 | "吃这个三天瘦五斤""一周消除内脏脂肪" | 合理设置预期，强调"效果因人而异" |
| 以偏概全 | "XX食物是垃圾食品，千万不能吃" | 强调剂量和频率，"没有垃圾食品，只有垃圾吃法" |
| 忽略个体差异 | 同一套方案推荐给所有人 | 标注适用人群和禁忌（如糖尿病、痛风等特殊人群） |
| 缺乏数据 | 讲营养但没有具体热量/营养素数据 | 标注热量、三大营养素含量等关键数据 |

### 6.2 制作层面

- **食物拍摄不诱人**：健康餐也要注重摆盘和光线，避免"暗黑料理"感
- **步骤不清晰**：食谱教程需要明确食材用量、火候、时间
- **缺少营养标注**：每道菜/每餐应标注总热量和主要营养素
- **BGM和节奏**：食谱类内容适合温馨轻快BGM，科普类适合节奏稍快的解说

## 七、合规红线

### 7.1 医疗/保健品合规（最高优先级）

- **禁止**宣称食物/食谱可以"治疗""治愈""根治"任何疾病
- **禁止**宣称普通食品具有保健功能或药用功效
- **禁止**用"抗癌食物""降糖神器""溶栓食物"等医疗化表述
- 保健食品（蓝帽产品）必须标注"本品不能代替药物"
- 涉及疾病相关饮食建议须加"请遵医嘱""建议咨询专业营养师/医生"

### 7.2 食品广告合规

- **禁止**食品广告涉及疾病预防和治疗功能
- **禁止**使用"最新科学""最新技术""最先进"等绝对化用语
- **禁止**普通食品暗示减肥功效（"吃了就能瘦""燃脂食物"）
- 推荐食品/保健品需标注"广告"或"合作"
- 代餐产品不得宣传替代正常饮食

### 7.3 平台规则

- 抖音/小红书对减肥饮食类内容有专项审核，极端节食内容会被删除
- "催吐""过度节食""极端断食"等内容直接封号处理
- 健康科普内容鼓励标注创作者资质（营养师/医生认证）
- AI生成的营养建议需标注
- 涉及特殊人群（孕妇、儿童、慢性病患者）须特别注意安全提示

### 7.4 安全表达替换表

| 敏感表达 | 安全替代 |
|---------|---------|
| "排毒食物/刮油食物" | "富含膳食纤维、有助于促进代谢的食物" |
| "燃脂食物/负卡路里" | "热量较低、饱腹感强的食物" |
| "降血糖/降血脂食物" | "对血糖/血脂管理有辅助作用的饮食搭配（请遵医嘱）" |
| "吃这个就能瘦" | "合理搭配这类食物，有助于控制总热量摄入" |
| "抗癌食物" | "富含抗氧化营养素的食物" |
| "这个食物千万不能吃" | "这类食物建议控制食用频率和分量" |
| "7天瘦10斤的食谱" | "科学减脂建议每周减0.5-1斤，这份食谱帮你健康起步" |

## 八、AI Prompt 注入要点

```
【营养饮食赛道知识注入】
1. 核心价值：提供好吃+好做+科学的饮食方案，让用户觉得"健康饮食没那么难"
2. 钩子优先级：误区颠覆 > 食谱推荐 > 食材揭秘 > 效果见证
3. 道心四法映射：营养误区/配料表揭秘=明道，饮食记录/效果见证=动心，食谱教程/食材清单=启思，反常识观点/创意吃法=破局
4. 食谱类必须标注热量和主要营养素，步骤须具体可执行
5. 禁止：伪科学（排毒/刮油/负卡路里）、医疗功效宣称、极端饮食推荐、食品广告化表述
6. 涉及疾病相关饮食必须加"请遵医嘱"，标注适用人群和禁忌
7. 语气：像一个懂营养学的好朋友在分享，科学但不晦涩，实用但不说教
8. 情绪曲线：误区震撼→科学解释→方案给到→鼓励行动
9. 结尾引导："需要完整一周食谱的评论区留言""你最想学哪类减脂餐"
10. 触发情绪：恍然大悟感、掌控感、"原来可以这么吃"的惊喜感
```
