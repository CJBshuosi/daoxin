## 一、赛道概况

- 易经相关话题在抖音累计播放量超200亿，"国学智慧"大类中《易经》内容占比持续上升
- 传统文化类短视频用户偏好占比达43%，位列内容类型偏好首位
- 头部易经类账号粉丝量突破300万，知识付费转化率高于国学平均水平
- 变现模式：知识付费（易经课程/读书会）+ 图书带货 + 直播讲解 + 企业咨询培训
- 受众画像偏"高决策力人群"：创业者、管理者、中年转型者比例显著高于其他国学赛道
- **此赛道为"封建迷信"监管重灾区，"传统文化智慧"与"封建迷信"的边界极为敏感，合规要求极高**

```
"你以为{关于易经的常见误解}？其实{真相}。"
```

- "'阴阳五行'不是玄学，而是古人对世界运行规律最早的系统建模。"

```
"《易经》第{N}卦，说透了{人生阶段/困境}。"
```

- "'否极泰来'不是安慰，是《易经》第12卦揭示的宇宙规律。"

## 五、有效内容角度

### 5.1 卦理人生角度（核心流量角度）

**公式**：具体卦象 + 白话解读 + 人生阶段映射 + 行动建议

```
选题示例："乾卦六爻：你的人生走到了哪一步？"
结构：简介卦象 → 逐爻对应人生阶段 → 你当前处于哪个阶段 → 对应的行动建议 → 金句收尾
```

**优势**：人人可代入，互动率极高（"我在第三爻"）

### 5.2 决策智慧角度

**公式**：现代决策困境 + 易经思维框架 + 具体分析方法 + 案例验证

```
选题示例："做重大决定前，用《易经》的三个问题检验"
结构：抛出决策痛点 → 介绍易经"时、位、中"框架 → 逐一拆解 → 实际案例 → 总结
```

**优势**：实用性极强，收藏率高，吸引高价值用户

### 5.3 文化溯源角度

**公式**：日常现象/习俗 + 追溯到易经源头 + 科学/哲学解读

```
选题示例："中国人为什么讲究'天时地利人和'？源头在《易经》"
结构：日常现象 → 追溯文化源头 → 《易经》的相关论述 → 背后的哲学道理 → 启发
```

**优势**：文化自豪感强，社交货币价值高

### 5.4 古今对照角度

**公式**：现代商业/管理概念 + 易经中的对应思想 + 深层关联解读

```
选题示例："VUCA时代的应对策略，《易经》3000年前就写好了"
结构：现代概念引入 → 易经的对应思想 → 两者深层共通的逻辑 → 可借鉴的方法论
```

**优势**：跨界新颖，打破"易经=老旧"的刻板印象

### 5.5 环境空间角度（需极度注意合规）

**公式**：空间/环境问题 + 古人的科学认知 + 现代科学佐证 + 实用建议

```
选题示例："古人为什么讲'坐北朝南'？这是环境科学，不是迷信"
结构：抛出习俗 → 古人的观察（采光/通风/地磁）→ 现代科学验证 → 可实践的建议
```

**优势**：实用性强，但必须去迷信化，用科学语言解释

## 六、常见错误与避坑

### 6.1 内容层面

| 错误 | 具体表现 | 正确做法 |
|------|---------|---------|
| 算命化 | "看看你属什么命"、"你的卦象显示……" | 讲哲理不讲命理，讲规律不讲算卦 |
| 迷信化 | "这样摆放能旺财"、"这个方位不吉利" | 用环境科学/空间心理学的语言重新包装 |
| 过度神化 | "易经预测了一切"、"古人什么都知道" | 客观看待，承认时代局限，突出哲学智慧价值 |
| 碎片化 | 只讲零散知识点，没有体系感 | 建立"入门→进阶→应用"的内容结构 |
| 太专业 | 天干地支、纳甲装卦等复杂术语 | 用大白话讲大智慧，术语即用即解释 |
| 标题党 | "看完改变命运"、"准得可怕" | 内容价值要配得上钩子的承诺 |
| 张冠李戴 | 将非易经内容归结为易经 | 引用必须准确标注出处，区分《周易》和后世注解 |

### 6.2 风格层面

- **忌"大师"感**：不要营造"半仙"人设，用学者/文化爱好者的人设
- **忌故弄玄虚**：越是深刻的道理，越要用简单的话说
- **忌功利化**：不要把《易经》变成"发财秘笈"
- **忌性别偏见**：古代的阴阳观不等于"男尊女卑"，注意现代价值观
- **忌绝对预测**：不要说"一定会发生"，改为"古人认为这种规律值得参考"

## 七、合规红线（最重要章节）

### 7.1 "传统文化智慧" vs "封建迷信" 的边界

这是易经风水赛道最核心的合规问题。平台判定标准：

| 维度 | 传统文化智慧（允许） | 封建迷信（禁止） |
|------|-------------------|----------------|
| 内容本质 | 讲哲学思想、辩证思维、生活智慧 | 讲算命、占卜、预测吉凶、改运 |
| 表达方式 | "古人认为……"、"这种思维方式……" | "你的命是……"、"必须这样做否则……" |
| 因果关系 | 讲规律、讲概率、讲思维框架 | 讲因果报应、宣扬超自然力量 |
| 空间环境 | 采光通风朝向的科学道理 | "这个位置旺财"、"那个方位犯煞" |
| 数字符号 | 文化溯源和象征意义解读 | 数字吉凶、号码运势 |
| 人物命运 | 历史人物的选择与智慧 | 面相手相、生辰八字、星座算命 |

### 7.2 绝对禁止的内容类型

```
1. 算命/占卜/测字/抽签/塔罗——任何形式的"预测个人命运"
2. 看风水——宣称空间布局影响运势/财运/健康（超自然因果）
3. 看面相/手相/骨相——通过外貌判断命运
4. 生辰八字/紫微斗数——任何基于出生时间的命运推算
5. "化解"/"破解"——宣称某种方法可以改变运势
6. 法器/护身符/开光物品的推广
7. 收费算卦/看相/择日等付费迷信服务引流
```

### 7.3 平台审核重点

| 审核维度 | 具体规则 |
|---------|---------|
| 关键词触发 | "算命"、"占卜"、"风水布局"、"旺财"、"犯太岁"、"化煞"等词直接触发审核 |
| 语义识别 | 即使不用敏感词，AI会通过上下文判断是否暗示迷信——如"床头朝这个方向睡得好"可能被判定 |
| 谐音替代无效 | "feng水"、"算m1ng"等替代写法已被平台AI识别 |
| 商品关联 | 如果橱窗有手串/罗盘/风水摆件，内容审核标准更严 |
| 直播更严 | 易经类直播被重点监管，在线人数建议控制在200以内 |
| 视觉元素 | 罗盘、八卦图、符咒等画面元素可能触发审核 |

### 7.4 安全表达替换表

| 危险表达 | 安全替代 |
|---------|---------|
| "风水好/风水不好" | "古人的空间智慧" / "环境布局的讲究" |
| "旺财/旺运" | "营造舒适的生活环境" / "提升居住品质" |
| "犯太岁/犯煞" | 避免使用，无安全替代 |
| "算一卦" | "从卦象的哲理来看" / "用易经思维分析" |
| "你的命格/命盘" | 避免使用，改为"你当前的处境/阶段" |
| "吉日/凶日" | "古人选择时机的考量" / "传统习俗的由来" |
| "镇宅/辟邪" | "传统文化中的美好寓意" |
| "改命/转运" | "改变思维方式" / "调整人生策略" |
| "五行缺X" | 避免使用，改为"五行思想的哲学意义" |
| "这个方位不好" | "古人在空间设计上的考量是……" |

### 7.5 安全内容定位公式

```
安全定位 = 《易经》哲学思想 + 辩证思维 + 决策智慧 + 文化溯源
危险定位 = 《易经》占卜算命 + 风水布局 + 命运预测 + 吉凶判断
```

**核心原则**：把《易经》当作"中国最古老的哲学经典和思维工具"来分享，而不是当作"占卜算命的神秘之书"来传播。讲"变化的规律"，不讲"命运的预测"。

### 7.6 高风险选题（建议回避或极度谨慎）

- 任何涉及"测一测你的XX"的互动选题
- 涉及具体风水布局指导的内容
- 涉及"择日"、"黄道吉日"等内容
- 涉及生肖运势、流年运势的预测
- 涉及面相/手相/骨相的内容
- 涉及"化太岁"、"犯太岁"等民间说法
- 涉及具体风水商品（罗盘/摆件/符咒）的推荐

### 7.7 免责话术模板

```
"以上为《易经》哲学思想的文化解读，旨在分享传统智慧，不涉及任何占卜或迷信。"
"本内容仅为传统文化科普，不构成任何预测或指导建议。"
"《易经》的价值在于其哲学思想和辩证思维，本内容仅做文化分享。"
```

### 8.2 成功要素

1. **人设定位**：学者/文化传播者，而非"大师"/"先生"，避免"半仙"联想
2. **视觉规范**：太极/水墨/竹简等元素点缀即可，避免罗盘/符咒等敏感视觉
3. **语言风格**：理性、有格局感，像一位懂哲学的朋友在聊天
4. **系列化**：如"易经64卦人生课"、"《周易》逐篇白话讲"
5. **金句密度**：每条至少1个可记忆的哲理金句
6. **互动设计**："你觉得自己在哪一爻？"引发评论区讨论
