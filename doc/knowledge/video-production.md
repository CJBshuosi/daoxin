## 1. 视频类型与制作模式

### 1.1 口播类（Talking Head）

**适用场景：** 知识分享、产品测评、观点输出、带货种草

#### 机位与角度
| 机位方案 | 说明 | 适用场景 |
|---------|------|---------|
| 单机位正面 | 摄像头正对面部，平视角度 | 知识科普、日常分享 |
| 单机位微侧（15°-30°） | 略偏一侧，更有电影感 | 个人IP、高端感内容 |
| 双机位切换 | 正面+侧面45°交替剪辑 | 长口播（>60秒），增加画面丰富度 |
| 俯拍机位 | 从上方拍桌面操作 | 教程类口播+手部演示 |

#### 构图规范
- **景别选择：** 以中景（胸部以上）或近景（肩部以上）为主
- **九宫格法则：** 人物眼睛放在上1/3线位置
- **头部留白：** 头顶保留画面高度的1/8-1/6空间
- **视线方向留白：** 人物看向一侧时，该侧留出更多空间
- **人物位置：** 居中或偏左/右1/3处（为字幕和贴纸留空间）

#### 背景设置
- **纯色背景：** 简洁干净，适合知识类（推荐浅灰、米白、浅蓝）
- **场景化背景：** 书架、工作台等，增加专业感和信任度
- **虚化背景：** 使用大光圈或手机人像模式，突出人物主体
- **核心原则：** 背景不能有杂物干扰，不能比人物更抢眼

#### 布光方案
| 方案 | 灯光配置 | 效果 |
|------|---------|------|
| 基础方案 | 1盏环形补光灯（正面） | 面部均匀亮度，适合新手 |
| 进阶方案 | 主灯（45°侧前方）+ 辅灯（另一侧补阴影） | 面部有层次感 |
| 专业方案 | 主灯 + 辅灯 + 轮廓灯（背后）+ 背景灯 | 人物与背景分离，纵深感强 |
| 自然光方案 | 面向窗户坐（窗户作为主光源） | 柔和自然，适合生活类内容 |

**布光要点：**
- 主灯亮度 > 辅灯亮度，比例约为2:1
- 避免顶光直射（会在眼眶下产生阴影）
- 背景灯制造纵深感，将人物与背景分离
- 口播类视频中人脸不能有明显阴影

#### 收音建议
- 使用领夹式/耳夹式麦克风，减少环境噪音
- 距离嘴部20-30cm为最佳收音距离
- 环境降噪：关闭空调、关窗、远离噪音源

### 1.2 图文类（Image + Text）

**适用场景：** 知识清单、产品对比、攻略合集、情感语录

#### 图片选择原则
- **高清优先：** 分辨率不低于1080p
- **风格统一：** 同一视频内图片色调、风格保持一致
- **信息密度：** 每张图片承载1个核心信息点
- **版权安全：** 使用无版权图库或自拍素材

#### 文字动画效果
| 效果类型 | 说明 | 适用场景 |
|---------|------|---------|
| 打字机效果 | 文字逐字出现 | 悬念感、叙事类 |
| 弹入/飞入 | 文字从画面外滑入 | 重点标注、列表展示 |
| 缩放弹出 | 文字从小变大弹出 | 强调、惊喜感 |
| 渐显 | 文字淡入出现 | 文艺风、情感类 |
| 描边/高亮 | 关键词变色或加描边 | 重点强调 |

#### 转场设计
- **滑动转场：** 图片左右/上下滑动切换，节奏感强
- **缩放转场：** 从细节放大到全景，或反之
- **遮罩转场：** 圆形/方形遮罩展开，适合列表类
- **翻页/3D翻转：** 适合产品对比类内容
- **统一原则：** 同一视频内转场风格保持一致，不超过2种转场类型

#### 图文视频节奏
- 每张图片停留2-4秒
- 文字出现时间：比画面切换延迟0.3-0.5秒（先看图再看字）
- 关键信息加粗/变色，停留时间延长至4-5秒
- BGM节拍与图片切换同步

### 1.3 混剪类（Mashup / Compilation）

**适用场景：** 影视解说、产品合集、盘点类、情绪向内容

#### 素材选取原则
- **主题一致性：** 所有素材围绕同一主题/情绪
- **画质一致性：** 尽量选择相近画质的素材
- **多样性：** 景别、角度、场景要有变化
- **版权合规：** 标注素材来源，避免原片搬运

#### 剪辑技巧
| 技巧 | 说明 | 效果 |
|------|------|------|
| 踩点剪辑 | 画面切换与BGM节拍同步 | 强烈节奏感 |
| 速度变化 | 关键画面慢放+普通画面正常/快放 | 重点突出 |
| 交叉剪辑 | 两条线索交替展示 | 对比感/悬念 |
| 跳切 | 同一主体不同时刻直接切换 | 时间流逝感 |
| 匹配剪辑 | 前后画面形状/动作/颜色相似 | 流畅过渡 |

#### 混剪节奏控制
- 开头（0-3秒）：最精彩/冲击力最强的画面
- 中段：按BGM节奏排列，高潮段落加快剪辑率
- 结尾：情绪回落或再次高潮+引导互动
- 平均镜头时长：1.5-3秒（快节奏）、3-5秒（慢节奏）

### 1.4 情景剧类（Skit / Scenario）

**适用场景：** 剧情演绎、产品植入、生活场景还原、搞笑段子

#### 场景设置
- **场景选择：** 贴合剧情主题，室内场景注意布景完整性
- **道具准备：** 提前准备剧情所需的所有道具
- **服装换装：** 不同角色/时间线用服装区分

#### 拍摄要点
| 要素 | 要求 |
|------|------|
| 景别覆盖 | 至少包含3种景别（远/中/近或特写） |
| 正反打 | 对话场景使用正反打（过肩镜头） |
| 建立镜头 | 每个新场景先用全景/远景交代环境 |
| 反应镜头 | 对话中穿插听者的反应镜头 |
| 轴线规则 | 对话双方拍摄不要越轴（180度原则） |

#### 对话节奏
- 台词之间的停顿：0.5-1秒（紧凑感）
- 搞笑类：关键笑点前留0.5秒"静拍"（蓄力感）
- 反转类：反转前停顿1-2秒，增加期待感
- 每段对话结束后用反应镜头过渡

#### 情景剧常用镜头序列
```
建立镜头（全景/远景，交代环境）
→ 中景（人物出场/互动）
→ 近景/特写（表情、关键动作）
→ 过肩镜头（对话正反打）
→ 特写（情绪高潮/反转点）
→ 中景/全景（收尾）
```

### 1.5 Vlog类

**适用场景：** 日常记录、旅行分享、探店、沉浸式体验

#### 拍摄手法
| 手法 | 说明 | 适用场景 |
|------|------|---------|
| POV（第一视角） | 手持拍摄，模拟观众视角 | 探店、美食、开箱 |
| 自拍视角 | 面向自己拍摄，对镜说话 | 出行记录、心情分享 |
| 跟拍 | 第三人跟随拍摄 | 旅行、户外活动 |
| 延时/快放 | 长时间过程压缩 | 做饭过程、日出日落 |
| 升格（慢动作） | 关键瞬间放慢 | 美食入口、风景特写 |

#### 运镜技巧
- **推镜头：** 由远及近，逐渐揭示细节（适合美食特写、产品展示）
- **拉镜头：** 由近及远，展示全貌（适合环境交代、结尾收束）
- **摇镜头：** 固定位置水平/垂直旋转（适合全景展示、空间交代）
- **移镜头：** 水平移动拍摄（适合货架展示、场景过渡）
- **跟镜头：** 跟随被摄对象移动（适合人物行走、动态场景）
- **升降镜头：** 垂直移动（适合建筑、高低落差场景）

#### Vlog自然转场技巧
- **手部遮挡：** 用手遮镜头 → 切换场景 → 手移开
- **快速甩镜头：** 向一个方向快速甩 → 切 → 从同方向甩入新场景
- **物体遮挡：** 走过门框/柱子时切换
- **相似元素：** 前后画面中有相似颜色/形状的物体做过渡
- **声音转场：** 下一个场景的声音提前出现（J-cut）

## 2. 镜头语言模板

### 2.1 开头Hook镜头（黄金前3秒）

**核心目标：** 3秒内留住观众，降低跳出率

| Hook类型 | 镜头设计 | 文案配合 | 适用内容 |
|---------|---------|---------|---------|
| **悬念型** | 特写 → 快速拉开，或模糊 → 清晰 | "你绝对想不到..." / "千万别..." | 反转类、揭秘类 |
| **利益型** | 产品/效果特写 | "学会这招，立省XXX" / "这是全网最全的..." | 教程类、种草类 |
| **冲突型** | 冲突画面（摔东西、惊讶表情） | "我被XX坑惨了" / "所有人都错了" | 测评类、观点类 |
| **数字型** | 数字大字报式文字覆盖 | "3个方法" / "月入5万的秘密" | 干货类、攻略类 |
| **身份标签型** | 人物中景+文字标签 | "30岁还单身的看过来" / "新手妈妈必看" | 精准人群定位 |
| **视觉冲击型** | 高速运动/极端特写/反常画面 | （文字辅助，或纯画面冲击） | 美食、旅行、创意类 |
| **对比型** | 分屏或前后对比画面 | "用之前 vs 用之后" | 测评、教程、变装 |
| **直入主题型** | 人物近景，直视镜头 | "今天教你XXX" / "这个方法亲测有效" | 知识类、口播类 |

#### 开头镜头节奏模板
```
【方案A - 悬念开头】
第1秒：特写/异常画面（制造好奇心）
第2秒：快速切到人物/场景（建立context）
第3秒：文字弹出核心Hook文案

【方案B - 直给开头】
第1秒：人物近景+开口说话（直接抛出价值点）
第2-3秒：画面切B-roll或图文辅助

【方案C - 视觉冲击开头】
第1秒：高冲击力画面（慢动作/快剪/特效）
第2秒：正常速度展示主题
第3秒：文字+语音揭示内容方向
```

### 2.2 转场类型与使用场景

| 转场类型 | 技术说明 | 最佳使用场景 | 注意事项 |
|---------|---------|-------------|---------|
| **硬切（Cut）** | 直接切换，无过渡效果 | 快节奏内容、日常剪辑 | 最常用，占80%以上 |
| **跳切（Jump Cut）** | 同一镜头中去掉中间片段 | 口播去废话、时间流逝 | 口播类最常用的剪辑方式 |
| **叠化（Dissolve）** | 前后画面交叉淡入淡出 | 时间流逝、回忆、梦境 | 时长0.5-1秒 |
| **淡入黑场（Fade）** | 画面渐黑后渐亮 | 段落分隔、开头结尾 | 节奏会变慢，慎用于快节奏内容 |
| **滑动/推移（Wipe）** | 新画面从一侧推入 | 图文类、对比类、列表展示 | 方向保持一致 |
| **缩放转场（Zoom）** | 画面放大到模糊后切入新画面 | Vlog场景切换、旅行类 | 配合运镜使用效果更好 |
| **遮挡转场** | 利用物体遮挡切换 | Vlog、情景剧 | 需拍摄时就设计好 |
| **匹配转场** | 前后画面形状/颜色/动作相似 | 创意混剪、品牌宣传 | 需提前规划素材 |
| **声音转场（J-Cut/L-Cut）** | 声音先于/晚于画面切换 | 所有类型均可用 | 增加流畅感的高级技巧 |

### 2.3 常用镜头序列模板

#### 产品种草类
```
1. 开头特写：产品最吸引人的细节/使用效果（1-2秒）
2. 中景：人物+产品同框，开始讲述（3-5秒）
3. 特写：产品细节展示（2-3秒）
4. 中景/近景：使用过程演示（5-10秒）
5. 特写：使用前后对比/效果展示（3-5秒）
6. 近景：总结推荐+引导互动（3-5秒）
```

#### 知识科普类
```
1. 近景：人物直视镜头，抛出问题/Hook（2-3秒）
2. 图文/B-roll：辅助解释第一个知识点（5-8秒）
3. 近景：回到人物，承上启下（2-3秒）
4. 图文/B-roll：辅助解释第二个知识点（5-8秒）
5. 近景：总结+金句收尾（3-5秒）
```

#### 探店Vlog类
```
1. 门头/招牌全景（1-2秒）
2. POV推门进入（1-2秒）
3. 环境摇镜头/移镜头（3-5秒）
4. 特写：菜单/亮点细节（2-3秒）
5. 中景：入座/点餐过程（3-5秒）
6. 特写：食物/产品特写，升格拍摄（3-5秒）
7. 近景：试吃/体验反应（3-5秒）
8. 近景：总结评价+推荐语（3-5秒）
```

#### 穿搭展示类
```
1. 全身镜全景/远景：整体造型一览（2-3秒）
2. 中景：上半身细节（2-3秒）
3. 特写：配饰/面料/细节（1-2秒）
4. 全景：转身/走动展示（2-3秒）
5. 中景：搭配讲解/替换单品（3-5秒）
6. 全景：最终整体展示（2-3秒）
```

## 3. 剪辑节奏模板

### 3.1 核心节奏原则

**"3秒定律"（3秒惯调）：** 每3秒必须提供新信息或新的视觉刺激。超过3秒无新信息，观众注意力开始下降。

**节奏变化方法：**
1. 改变镜头剪辑率（加快/放慢切换频率）
2. 改变景别（远→近→特写→远）
3. 改变运镜方式（静→动→静）
4. 改变画面稳定性（稳定器→手持抖动→稳定器）
5. 改变播放速度（正常→慢放→加速）
6. 改变主客观镜头（旁观→POV→旁观）

### 3.2 不同时长的节奏模板

#### 15秒视频（抖音/快手黄金时长）
```
结构：Hook → 核心内容 → 收尾引导
时间分配：
  0-3秒  ：Hook（强冲击开头，1-2个镜头）
  3-10秒 ：核心内容（主要信息，3-4个镜头）
  10-15秒：收尾（总结/反转/引导点赞，1-2个镜头）

镜头数量：6-8个
平均单镜头时长：1.5-2.5秒
剪辑节奏：快（全程紧凑，无废帧）
```

#### 30秒视频
```
结构：Hook → 铺垫 → 核心 → 收尾
时间分配：
  0-3秒  ：Hook（黄金3秒）
  3-10秒 ：铺垫/背景交代
  10-22秒：核心内容展开
  22-30秒：总结/CTA

镜头数量：10-15个
平均单镜头时长：2-3秒
剪辑节奏：中快（可有1-2次节奏变化）
```

#### 60秒视频
```
结构：Hook → 问题/背景 → 解决方案/展开 → 高潮 → 收尾
时间分配：
  0-3秒  ：Hook
  3-15秒 ：问题提出/背景铺垫
  15-40秒：核心内容展开（可分2-3个要点）
  40-50秒：高潮/最强信息点
  50-60秒：总结+CTA

镜头数量：20-30个
平均单镜头时长：2-3秒
剪辑节奏：有快有慢（至少2-3次节奏起伏）
节奏曲线：中 → 稍慢 → 快 → 最快 → 中
```

#### 2分钟+视频
```
结构：Hook → 引入 → 主体（分段） → 高潮 → 总结
时间分配：
  0-5秒  ：Hook
  5-20秒 ：引入主题
  20-90秒：主体内容（每20-30秒一个段落/要点）
  90-110秒：高潮/最精华内容
  110-120秒+：总结+引导互动

镜头数量：40-60个
平均单镜头时长：2-4秒
剪辑节奏：波浪式（每个段落有自己的小高潮）
段落间标记：可用文字标题卡/转场音效分隔
```

### 3.3 口播类剪辑节奏专项

| 技巧 | 说明 | 操作方法 |
|------|------|---------|
| 跳切去废话 | 删掉停顿、重复、口误 | 直接跳切，保持说话连贯 |
| 景别切换 | 同一内容用不同景别 | 双机位拍摄或后期裁切放大 |
| 插入B-roll | 说到某内容时插入相关画面 | 辅助理解，打破单调 |
| 文字强化 | 关键词/数字以文字弹出 | 与语音同步出现 |
| 表情包/贴纸 | 增加趣味性 | 适当使用，不要过多 |
| 音效点缀 | 关键节点加音效 | 强调重点、制造节奏变化 |

### 3.4 音乐卡点剪辑

**基本原则：**
- 画面切换点与音乐节拍点（Beat）对齐
- 强拍处放重要画面切换，弱拍处放细微调整
- 高潮段落加快剪辑率，间奏段落放慢节奏

**操作步骤：**
1. 先选定BGM，标记所有节拍点
2. 将主要画面切换点对齐到强拍
3. 副画面/文字动画对齐到弱拍
4. 高潮段可使用"一拍一切"（每个节拍切一次）
5. 前奏段可使用"两拍一切"或"四拍一切"

**不同BPM的适用场景：**
| BPM范围 | 节奏感受 | 适用内容类型 |
|---------|---------|-------------|
| 60-80 | 舒缓、沉静 | 情感故事、知识科普、治愈系 |
| 80-100 | 轻快、舒适 | 日常Vlog、生活分享 |
| 100-120 | 活泼、欢快 | 美食、旅行、开箱 |
| 120-140 | 动感、激昂 | 运动、变装、快节奏混剪 |
| 140+ | 高能、炸裂 | 踩点混剪、高燃剪辑 |

### 3.5 文字叠加时机与规范

#### 文字出现时机
- 语音说出后**0.3-0.5秒内**出现对应字幕（人耳先听，眼再读）
- 关键词/数字可**与语音同步**弹出，甚至**提前0.2秒**出现（预期引导）
- 字幕停留时长 = 阅读时间 + 0.5秒缓冲（一般3-5字停留1.5秒，10字以上停留3秒）

#### 文字位置规范
| 位置 | 用途 | 注意事项 |
|------|------|---------|
| 画面上方1/5 | 标题/Hook文案 | 不遮挡人物头部 |
| 画面中央 | 大字报式强调 | 仅用于关键信息，停留1-2秒 |
| 画面下方1/4 | 常规字幕 | 避开平台水印和互动按钮区域 |
| 画面左右两侧 | 补充说明/标签 | 不遮挡人物面部 |

#### 文字样式原则
- **字体：** 无衬线字体为主（思源黑体、阿里巴巴普惠体等）
- **字号：** 关键词>标题>正文字幕，至少3档大小区分
- **颜色：** 白色字+黑色描边最通用；亮色底用深色字，暗色底用浅色字
- **动画：** 同一视频内文字动画风格统一，不超过2种动画效果
- **数量：** 单屏文字不超过20字（观众3秒内能读完）

## 4. 音乐与音效

### 4.1 BGM选择原则

#### 按内容情绪匹配

| 内容类型 | 推荐BGM风格 | BPM参考 | 示例场景 |
|---------|------------|---------|---------|
| 知识科普 | 轻快电子/钢琴轻音乐 | 80-110 | 干货讲解、技巧分享 |
| 美食类 | 欢快ukulele/爵士/轻快流行 | 100-120 | 做饭过程、探店试吃 |
| 情感故事 | 钢琴曲/弦乐/舒缓民谣 | 60-80 | 讲述经历、情感分享 |
| 搞笑类 | 轻快弹跳/喜剧配乐 | 100-130 | 搞笑段子、整蛊 |
| 旅行类 | 民族风/轻摇滚/Indie Pop | 90-120 | 旅行记录、风景展示 |
| 时尚穿搭 | 电子/R&B/流行舞曲 | 110-140 | 穿搭展示、变装 |
| 励志/运动 | 激昂摇滚/电子/Hip-hop | 120-150 | 健身、挑战、励志故事 |
| 治愈/生活 | Lo-fi/轻音乐/自然音 | 70-90 | 日常Vlog、慢生活 |
| 高燃混剪 | Epic/Trailer Music/EDM | 130-160 | 精彩瞬间合集 |

#### BGM使用核心技巧

1. **音量控制：**
   - 纯BGM画面：音量70-100%
   - 有人声画面：BGM降至20-30%（人声为主）
   - 重点强调处：BGM可瞬间拉高再回落

2. **新手选曲建议：**
   - 优先选择**纯音乐**（不受歌词干扰，包容性强）
   - 使用平台内置音乐库（剪映/抖音/小红书自带，免版权风险）
   - 跟随热门BGM可获得额外流量推荐

3. **BGM结构利用：**
   - 前奏 → 对应视频铺垫段
   - 副歌/高潮 → 对应视频核心/高潮段
   - 间奏 → 对应过渡段
   - 结尾 → 对应视频收尾

### 4.2 人声方案选择

| 方案 | 优势 | 劣势 | 适用场景 |
|------|------|------|---------|
| 真人出镜口播 | 信任感强、有个人IP感 | 需要表现力、对形象有要求 | 个人IP、带货、测评 |
| 真人旁白（不出镜） | 亲切感强、制作门槛低 | 缺少视觉焦点 | 美食教程、图文解说 |
| AI配音 | 效率高、声音稳定 | 缺少情感、辨识度低 | 资讯类、大批量内容 |
| 纯文字+BGM（无人声） | 制作最简单 | 信息传达效率低 | 情绪向、图文类 |

**人声与BGM配比原则：**
- 口播/解说类：人声70% + BGM30%
- Vlog类：人声50% + BGM40% + 环境音10%
- 纯画面类：BGM80% + 环境音/音效20%

### 4.3 音效使用指南

#### 常用音效类型及场景

| 音效类型 | 具体音效 | 使用场景 |
|---------|---------|---------|
| **转场音效** | Whoosh（嗖声）、Swoosh | 画面切换时增加流畅感 |
| **强调音效** | Ding（叮）、Pop（弹）、Boom | 文字弹出、重点强调、数字出现 |
| **情绪音效** | 笑声、惊叹、鼓掌 | 搞笑、惊喜、认可等情绪点 |
| **环境音效** | 键盘声、翻书声、咖啡杯声 | 场景氛围营造 |
| **节奏音效** | 倒计时、心跳、钟表 | 悬念、紧张感营造 |
| **反馈音效** | 现金声、成功提示音 | 利益点、成果展示 |
| **卡通音效** | 弹簧声、滑稽声 | 搞笑、轻松类内容 |

#### 音效使用原则
1. **少而精：** 每10秒视频不超过2-3个音效点
2. **与画面同步：** 音效出现时机与画面动作/文字对齐
3. **音量适中：** 音效音量略高于BGM，但不高于人声
4. **风格统一：** 同一视频内音效风格保持一致
5. **J-Cut技巧：** 下一个场景的声音提前0.5-1秒出现，增加流畅感
6. **L-Cut技巧：** 前一个场景的声音延续到下一个画面，增加连贯性

## 5. 平台差异化指南

### 5.1 抖音（Douyin）

| 参数 | 规范 |
|------|------|
| **画面比例** | 9:16竖屏（强制推荐） |
| **推荐分辨率** | 1080×1920 |
| **推荐时长** | 15-60秒（新手建议15-30秒） |
| **最长时长** | 15分钟（普通用户），60分钟（开通长视频权限） |

#### 算法偏好与制作要点
- **黄金3秒至上：** 前3秒跳出率是核心指标，开头必须有强Hook
- **完播率优先：** 视频完播率是最重要的推荐指标
- **内容>一切：** 内容质量占推荐权重的90%
- **节奏要快：** 平均镜头时长2-3秒，信息密度高
- **互动引导：** 结尾引导点赞、评论、关注，提升互动率
- **热门BGM加成：** 使用当前热门BGM可获得额外推荐
- **文字要大：** 竖屏字幕字号要大，确保手机屏幕清晰可读

#### 抖音内容禁区
- 避免明显的硬广感（自然种草>直接推销）
- 避免搬运/重复内容（查重机制严格）
- 避免低画质/模糊素材
- 封面需有文字标题（提升点击率）

---

### 5.2 小红书（Xiaohongshu / RED）

| 参数 | 规范 |
|------|------|
| **画面比例** | 3:4竖屏（推荐）、1:1方形、16:9横屏均可 |
| **推荐分辨率** | 1080×1440（3:4）或 1080×1920（9:16） |
| **推荐时长** | 1-3分钟（可较长） |
| **最长时长** | 15分钟 |

#### 算法偏好与制作要点
- **封面为王：** 双瀑布流展示，封面决定点击率（最关键因素）
- **标题党有效：** 封面大字报+吸引力标题组合
- **精致审美：** 画面色调统一、滤镜精致、有质感
- **CES评分机制：** 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
- **收藏导向：** 干货/教程类内容高收藏率，权重高于点赞
- **搜索流量大：** 标题和内容中嵌入搜索关键词
- **节奏可以慢：** 相比抖音，用户耐心更长，可以更详细地展开

#### 小红书视频风格特点
- 画面色调偏暖/清新/高级感
- 文字排版讲究，常用小标题分段
- BGM偏文艺/轻快/小众
- Vlog感和生活感强，避免过度商业化
- 可使用横屏（16:9），适合教程/对比类

---

### 5.3 微信视频号（Channels）

| 参数 | 规范 |
|------|------|
| **画面比例** | 6:7（推荐）、9:16竖屏、16:9横屏均可 |
| **推荐分辨率** | 1080×1260（6:7）|
| **推荐时长** | 1-3分钟（可更长） |
| **最长时长** | 30分钟 |

#### 算法偏好与制作要点
- **社交推荐为核心：** 朋友点赞/转发是最大流量来源（私域撬动公域）
- **完播分段考核：** 前3秒跳出率需<45%，30秒留存率≥60%
- **用户偏成熟：** 30-50岁用户占比高，内容可以更深度
- **转发权重高：** "转发到朋友圈/群聊"是关键互动指标
- **节奏可更慢：** 用户接受度更高，可以娓娓道来
- **可用横屏：** 横屏16:9在视频号中表现可接受

#### 视频号内容调性
- 更偏向"有价值""正能量""实用性"
- 知识类、职场类、生活感悟类效果好
- 避免过于年轻化/网感过重的风格
- 可以更长更详细，用户有耐心看完
- 鼓励引导用户转发到微信群和朋友圈

## 附录：常用术语速查

| 术语 | 含义 |
|------|------|
| Hook | 开头钩子，吸引观众继续观看的元素 |
| CTA | Call To Action，行动号召（点赞/评论/关注/购买） |
| B-roll | 辅助画面，用于补充说明主画面内容 |
| 完播率 | 观众看完整个视频的比例 |
| 跳出率 | 观众在前几秒就划走的比例 |
| BPM | Beats Per Minute，每分钟节拍数 |
| 踩点 | 画面切换与音乐节拍同步 |
| J-Cut | 下一场景声音提前于画面出现 |
| L-Cut | 前一场景声音延续到下一画面 |
| 正反打 | 对话场景中两人交替拍摄 |
| 越轴 | 摄像机越过180度线，导致方向混乱 |
| 景别 | 镜头取景范围：远景>全景>中景>近景>特写 |
| 九宫格构图 | 画面分成9格，主体放在交叉点 |
| POV | Point of View，第一人称视角 |
| 升格 | 高帧率拍摄后正常播放，呈现慢动作效果 |
| 硬切 | 无过渡效果的直接画面切换 |
| 跳切 | 同一镜头去掉中间部分直接拼接 |
