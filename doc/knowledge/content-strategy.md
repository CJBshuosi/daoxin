# 道心文案 · 内容策略方法论（Prompt 知识库）

> 适用平台：抖音、小红书、视频号
> 用途：道心文案 AI Prompt 知识库，提供系统化的短视频内容策略框架
> 本框架为道心文案原创，基于短视频创作通用方法论整理

---

## 一、道心四法 · 内容策略框架

短视频内容创作的本质，是在有限时间内完成"吸引→触动→行动"的闭环。道心文案将内容策略归纳为**四法**，每法下设具体方向，覆盖所有可能的内容角度。

四法之间不是互斥关系，一条优秀的短视频往往会融合多法，但主线应明确。

---

### 1.1 【明道】洞见之法 — 以认知落差驱动传播

> 核心逻辑：**让观众觉得"原来如此"**。揭示真相、打破误区、展现深层规律，用认知落差制造传播势能。

适用目标：完播率、收藏、涨粉

#### ① 认知颠覆

- **定义**：打破目标用户的既有认知，揭示被忽略或误解的真相
- **适用场景**：用户群体中存在广泛的认知偏差或信息不对称
- **选题角度示例**：
  - "你以为的XX其实完全是错的，真相是……"
  - "这个常识骗了你很多年"
  - "XX行业不会告诉你的3个真相"
- **底层心理**：求知欲 + 认同欲 — 人渴望"正确答案"，对"被蒙蔽"有天然警惕
- **执行要点**：必须有扎实论据；对比"错误认知 vs 正确认知"形成反差；带"揭秘"感但不哗众取宠

#### ② 深层规律

- **定义**：揭示某个现象背后的底层逻辑、运行规律或因果关系
- **适用场景**：用户对某个领域感兴趣但停留在表层理解
- **选题角度示例**：
  - "为什么90%的人做XX都失败了？根源在这"
  - "古人早就看透了这个规律"
  - "XX的底层逻辑，只有三个字"
- **底层心理**：求知欲 — 人天然追求"为什么"和"怎么回事"
- **执行要点**：用简洁的模型或公式概括复杂规律；配合具体案例让抽象变具象

#### ③ 趋势洞察

- **定义**：围绕领域内的新变化、新趋势进行解读和预判
- **适用场景**：行业变化、热点事件、政策调整、季节转换
- **选题角度示例**：
  - "这个变化大多数人还没注意到"
  - "接下来3个月，做XX的人要注意了"
  - "今年最大的变化已经开始了"
- **底层心理**：求知欲 + 生存欲 — FOMO（错失恐惧），人害怕掉队
- **执行要点**：时效性是核心；解读角度要独特，不做信息搬运工；强调"这意味着什么"

#### ④ 数据揭秘

- **定义**：用数据、研究、权威信源来支撑观点，增强说服力
- **适用场景**：需要建立专业感和权威感的内容
- **选题角度示例**：
  - "研究了1000条爆款视频，我发现了这个规律"
  - "这组数据颠覆了我的认知"
  - "XX机构最新报告，揭示了一个残酷真相"
- **底层心理**：求知欲 + 认同欲 — 数据天然有说服力，让人信服
- **执行要点**：数据来源要可信；用可视化或对比让数据生动；不堆砌数据，突出一个核心发现

---

### 1.2 【动心】共鸣之法 — 以情感共振驱动互动

> 核心逻辑：**让观众觉得"说的就是我"**。触动情感、引发共鸣、建立连接，用情绪共振驱动点赞评论。

适用目标：点赞、评论、转发

#### ① 情感共鸣

- **定义**：触达用户内心深层情感 — 亲情、乡愁、遗憾、成长等普世主题
- **适用场景**：节日节点、人生阶段转变、社会情绪高涨时
- **选题角度示例**：
  - "30岁以后才懂的道理"
  - "那些说出来矫情但确实很难受的瞬间"
  - "有些话，只有经历过的人才懂"
- **底层心理**：亲近欲 + 认同欲 — 人需要情感连接，通过他人的故事确认自己的感受被理解
- **执行要点**：真实感第一位；细节决定共鸣深度；避免刻意煽情；音乐和画面节奏配合

#### ② 身份认同

- **定义**：说出某个群体的心声、困境或骄傲，让用户产生强烈的身份认同
- **适用场景**：目标用户有明确的身份标签（年龄、职业、爱好、地域等）
- **选题角度示例**：
  - "只有XX人才能看懂这条视频"
  - "做XX这一行的，谁还没经历过这些"
  - "这就是我们XX人的日常"
- **底层心理**：认同欲 + 胜负欲 — 人需要群体归属感和身份确认
- **执行要点**：精准击中群体特征；用"我们"建立归属；避免排他性过强

#### ③ 态度表达

- **定义**：对某个话题或现象表达鲜明的立场和态度，引发讨论
- **适用场景**：社会热点、争议话题、价值观碰撞
- **选题角度示例**：
  - "我就是不认同XX，理由很简单"
  - "有些话不好听但是真话"
  - "这件事，我必须说两句"
- **底层心理**：认同欲 + 信仰欲 — 人渴望找到和自己立场一致的声音
- **执行要点**：态度鲜明但有论据支撑；不为了吵架而吵架；留出讨论空间让评论区活跃

#### ④ 故事感动

- **定义**：通过真实或典型的故事来传递价值观和情感
- **适用场景**：需要传递较深层次的价值观或情感时
- **选题角度示例**：
  - "我见过最厉害的人，做了这样一件事"
  - "那年冬天发生的事，我到现在都忘不了"
  - "一个普通人的故事，却让百万人泪目"
- **底层心理**：亲近欲 + 保护欲 — 好的故事让人产生代入感和情感投射
- **执行要点**：细节是故事的灵魂；节奏要有起伏；结尾要有余韵不要说教

### 1.3 【启思】价值之法 — 以实用价值驱动收藏

> 核心逻辑：**让观众觉得"学到了"**。提供实用方法、解决具体问题、传递可操作的知识。

适用目标：收藏、关注、完播

#### ① 干货输出

- **定义**：高信息密度的知识分享，让用户学到可立即使用的方法或信息
- **适用场景**：用户有明确的学习需求或信息获取需求
- **选题角度示例**：
  - "XX的5个方法，第3个最有效"
  - "花了3年总结的XX经验，全在这里了"
  - "新手最容易犯的3个错误"
- **底层心理**：求知欲 + 收藏欲 — 用户觉得有价值会收藏，收藏率直接影响推荐
- **执行要点**：信息密度要高，每3秒一个信息点；结构清晰（总分总、并列）；语言精炼不废话

#### ② 痛点解决

- **定义**：直接针对目标用户的具体痛点，提供解决方案
- **适用场景**：用户有明确的困扰或需求
- **选题角度示例**：
  - "一直困扰你的XX问题，其实很简单"
  - "XX总是做不好？试试这个方法"
  - "解决了这个问题，你的XX会完全不一样"
- **底层心理**：生存欲 + 享受欲 — 人天然被能解决自己问题的内容吸引
- **执行要点**：精准描述痛点让用户"对号入座"；方案要具体可执行；有before/after对比效果更好

#### ③ 经验复盘

- **定义**：分享自己或他人的实践经验、成功/失败案例的总结
- **适用场景**：用户在某个领域刚起步或正在探索
- **选题角度示例**：
  - "做了XX一年，我最大的感悟是这个"
  - "如果重来一次，我绝对不会做这件事"
  - "从0到1的过程，踩过的坑都在这了"
- **底层心理**：求知欲 + 保护欲 — 人希望借鉴他人经验避免弯路
- **执行要点**：真实感比完美更重要；承认失败比炫耀成功更有吸引力；总结出可复用的原则

#### ④ 对比测评

- **定义**：对同类事物进行对比分析，帮用户做出判断或选择
- **适用场景**：用户面临选择困难或需要决策参考
- **选题角度示例**：
  - "A和B到底选哪个？实测告诉你答案"
  - "用了XX和XX，差距真的太大了"
  - "别再纠结了，看完这条你就知道怎么选"
- **底层心理**：求知欲 + 收藏欲 — 帮用户降低决策成本
- **执行要点**：公正客观（至少看起来公正）；对比维度要用户关心的；结论明确不含糊

---

### 1.4 【破局】创意之法 — 以新奇视角驱动破圈

> 核心逻辑：**让观众觉得"没想到还能这样"**。用意想不到的角度、跨界联想、创意表达来制造惊喜。

适用目标：转发、出圈、涨粉

#### ① 跨界联想

- **定义**：将两个看似不相关的领域进行创意连接，产生化学反应
- **适用场景**：内容同质化严重，需要差异化突破
- **选题角度示例**：
  - "用经济学原理解释XX"
  - "XX和XX居然有这样的共同点"
  - "当XX遇上XX，意想不到的事发生了"
- **底层心理**：求知欲 + 享受欲 — 新奇的连接让人兴奋，大脑天然奖励新发现
- **执行要点**：两个领域的连接要有真实的逻辑关系，不能强行牵扯；陌生感 + 合理性 = 最佳效果

#### ② 极端假设

- **定义**：提出一个极端的假设条件，在想象中探索可能性
- **适用场景**：需要激发好奇心和讨论度的内容
- **选题角度示例**：
  - "如果XX活在今天，会怎么样？"
  - "假如你只能带3样东西……"
  - "如果XX消失了，世界会变成什么样？"
- **底层心理**：求知欲 + 享受欲 — 假设性思考满足人的想象力需求
- **执行要点**：假设要足够大胆但不荒谬；展开过程要有逻辑；结论要有洞察不能烂尾

#### ③ 时空穿越

- **定义**：将古今、中外、过去未来进行对照或穿越式演绎
- **适用场景**：历史、文化、传统类内容的创意表达
- **选题角度示例**：
  - "古人如果看到今天的XX，会说什么？"
  - "100年前的人绝对想不到……"
  - "把XX放到古代，会是什么样？"
- **底层心理**：求知欲 + 享受欲 — 时空碰撞产生天然的趣味性和反差
- **执行要点**：时空对比要有具体细节；不能只是概念，要有场景化演绎；反差越大越有趣

#### ④ 逆向思维

- **定义**：从相反的角度思考问题，提出与主流观点相反的论点
- **适用场景**：某个领域有强势的主流观点，需要提供不同视角
- **选题角度示例**：
  - "大家都说XX好，但我劝你别碰"
  - "越努力越失败的真正原因"
  - "成功的人从不做这件事"
- **底层心理**：求知欲 + 胜负欲 — 反直觉的观点让人好奇，也激发讨论欲
- **执行要点**：逆向不是抬杠，必须有扎实的逻辑支撑；承认主流观点的合理性再提出不同角度

#### ⑤ 视角转换

- **定义**：从一个不寻常的视角来讲述故事或分析问题
- **适用场景**：常规角度已经被讲烂，需要新鲜感
- **选题角度示例**：
  - "如果XX会说话，它想告诉你什么"
  - "从XX的角度看，这件事完全不一样"
  - "你站在XX的位置想一想"
- **底层心理**：求知欲 + 亲近欲 — 新视角让人产生全新的理解和共情
- **执行要点**：视角要选得巧妙；代入感是关键；用第一人称增强沉浸感

## 二、选题方法论

### 2.1 选题四维评估模型

每个选题从以下四个维度评估，**至少满足两个才值得做**：

| 维度 | 定义 | 判断标准 | 对应数据 |
|---|---|---|---|
| **有趣** | 能引发好奇或愉悦 | 看到标题会想点进来吗？ | 点击率 |
| **有共鸣** | 能触动情感或引发认同 | 看完会想点赞或评论吗？ | 互动率 |
| **有价值** | 能提供信息或解决问题 | 看完会想收藏或转发吗？ | 收藏率 |
| **有美感** | 视觉或文字有审美价值 | 愿意分享给朋友看吗？ | 转发率 |

### 2.2 九大底层欲望映射

所有爆款内容都触达了至少一个人类底层欲望：

| 欲望 | 表现 | 典型选题方向 | 对应策略法 |
|---|---|---|---|
| **生存欲** | 健康、安全、生存 | 养生、避险、健康知识 | 启思·痛点解决 |
| **享受欲** | 快乐、舒适、美好 | 美食、旅行、生活方式 | 动心·情感共鸣 |
| **收藏欲** | 占有、囤积、备用 | 合集、清单、教程 | 启思·干货输出 |
| **亲近欲** | 被爱、归属、连接 | 家庭、爱情、友谊 | 动心·故事感动 |
| **胜负欲** | 竞争、超越、优越 | 对比、排名、挑战 | 启思·对比测评 |
| **认同欲** | 被理解、被认可 | 共鸣、身份、态度 | 动心·身份认同 |
| **信仰欲** | 意义、使命、信念 | 人生哲学、价值观 | 动心·态度表达 |
| **保护欲** | 保护他人、责任 | 育儿、公益、避坑 | 启思·经验复盘 |
| **求知欲** | 好奇、探索、理解 | 科普、揭秘、解读 | 明道·认知颠覆 |

### 2.3 选题角度生成技法

#### 技法一：矩阵交叉法
将「四法子方向」与「欲望」交叉，产生新角度：
```
认知颠覆 × 生存欲 = "你每天在做的这件事，其实正在损害健康"
跨界联想 × 收藏欲 = "用整理术的逻辑管理你的知识库"
```

#### 技法二：热点嫁接法
将当前热点与垂直赛道强行连接：
```
热点事件 + 赛道专业知识 = 蹭热点但有深度的选题
```

#### 技法三：用户痛点挖掘法
从目标用户的日常困境出发，反推选题：
```
用户抱怨 → 提炼痛点 → 匹配策略法 → 生成选题
```

#### 技法四：爆款逆向工程法
分析同赛道已有爆款，提取其策略法组合：
```
爆款视频 → 识别用了哪个策略法 → 换主题复用同策略
```

## 三、钩子公式（前3秒）

前3秒决定用户是否停留。6种钩子类型：

### ① 反常识钩子
打破已有认知，制造好奇
```
公式：常见认知 + "其实完全是错的" / "真相是……"
示例："喝热水养生？古人根本不这么做"
```

### ② 情感共鸣钩子
说出用户心里的话
```
公式：身份标签 + 共同经历/感受
示例："每个在外打拼的人，都欠父母一句话"
```

### ③ 故事悬念钩子
制造好奇和紧张感
```
公式：结果前置 / 冲突引入 / 神秘人物
示例："那天深夜，师父说了一句话，彻底改变了我"
```

### ④ 权威数据钩子
用数据或权威建立信任
```
公式：具体数字 + 出人意料的结论
示例："翻遍了《黄帝内经》387条养生法，真正有效的只有3条"
```

### ⑤ 利益承诺钩子
直接告诉用户能获得什么
```
公式：目标场景 + 具体收益
示例："学会这3句话，任何社交场合你都不会冷场"
```

### ⑥ 好奇缺口钩子
制造信息缺口，让人非看不可
```
公式：部分信息 + 悬念留白
示例："成功的人有一个共同习惯，说出来你可能不信"
```

## 四、内容结构模型（8种）

### ① SCQA 模型（万能型）
```
Situation（场景）→ Complication（冲突）→ Question（问题）→ Answer（答案）
适用：几乎所有内容类型
```

### ② 故事弧线模型
```
日常 → 转折/冲突 → 挣扎 → 顿悟 → 新状态
适用：故事型、感悟型内容
```

### ③ 对比反转模型
```
常见认知 → 反面证据 → 真正答案 → 行动建议
适用：认知颠覆型内容
```

### ④ 渐进清单模型
```
总论 → 第1点 → 第2点 → 第3点 → 总结升华
适用：干货输出、经验分享
```

### ⑤ PREP 模型（观点型）
```
Point（观点）→ Reason（理由）→ Example（案例）→ Point（重申观点）
适用：态度表达、论证说服
```

### ⑥ 悬念连环模型
```
悬念1 → 部分揭示 → 新悬念 → 部分揭示 → 最终揭示
适用：故事悬念、连续剧式内容
```

### ⑦ 痛点打击模型
```
描述痛点场景 → 分析原因 → 提出方案 → 效果展示
适用：痛点解决、产品推荐
```

### ⑧ FIRE 模型（互动型）
```
Fact（事实/现象）→ Interpretation（解读）→ Reaction（引发反应）→ Ending（引导行动）
适用：热点评论、争议话题
```

## 五、结尾引导模式（CTA）

### 按目标分类：

**引导点赞：**
- 认同式："觉得说得对的，给我点个赞"
- 价值式："如果今天学到了一点东西，点个赞鼓励一下"
- 共鸣式："有同感的，双击屏幕"

**引导评论：**
