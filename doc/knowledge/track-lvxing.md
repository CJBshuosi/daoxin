# 旅行攻略赛道 垂直知识库

> 适用于「道心文案」AI Prompt 注入，提供旅行攻略赛道的选题、钩子、合规等垂直知识。基于「道心四法」框架驱动内容创作。

---

## 一、赛道概况

- 2026年文旅市场全面复苏，小红书发布《2026文旅兴趣出游种草指南》定义三大趋势
- 趋势一：兴趣成为核心出游动机——"为一场演出赴一座城""为一门手艺去一个村"
- 趋势二：境内外兴趣目的地全面崛起，烟火小城、边疆秘境被年轻人挖掘为新热点
- 趋势三：兴趣与四季绑定，形成时令性出游（春赏花、夏避暑、秋红叶、冬温泉）
- 内容策略从"爆款破圈"转向极致精细化，针对性打透微观场景、精准人群、小众兴趣圈层
- 深度旅行纪实取代打卡式vlog，注重"人与地的对话"
- AI成为文旅从业者的"基础生产力工具"，辅助行程规划和内容创作

## 二、受众画像

### 2.1 核心人群

| 人群标签 | 占比（估算） | 特征 |
|---------|------------|------|
| 22-30岁年轻白领 | 35% | 周末游/短途游主力，追求性价比和出片率 |
| 25-35岁情侣/新婚 | 20% | 浪漫目的地、小众打卡点、约会旅行 |
| 30-40岁亲子家庭 | 20% | 遛娃攻略、亲子酒店、安全便利是核心诉求 |
| 18-25岁学生党 | 15% | 穷游攻略、青旅推荐、Gap Year灵感 |
| 40-55岁银发旅行者 | 10% | 康养旅居、慢节奏旅行、文化历史深度游 |

### 2.2 用户痛点与需求

- **决策焦虑**：目的地太多选不好，怕踩雷浪费假期
- **攻略缺失**：去了不知道怎么玩，交通/住宿/路线一头雾水
- **预算有限**：想玩好但不想花冤枉钱，追求极致性价比
- **出片需求**：拍照好看是刚需，需要机位推荐和穿搭建议
- **避坑诉求**：景区宰客、网红滤镜骗局、旺季人挤人
- **独特体验**：不想千篇一律的打卡游，追求深度和差异化

### 2.3 用户心理触发词

```
绝了、此生必去、冷门宝藏、本地人才知道、人少景美、免费、避坑、拍照超出片、说走就走、治愈
```

## 三、爆款选题类型（映射道心四法）

### 3.1 选题类型总览

| 选题类型 | 道心四法归属 | 典型选题 | 爆款原因 |
|---------|-------------|---------|---------|
| 信息差揭秘型 | 明道·洞见 | "去三亚千万别住亚龙湾，本地人都去这个湾"、"90%的人不知道故宫有个免费花园" | 认知落差制造"涨知识"的传播动力 |
| 情感叙事型 | 动心·共鸣 | "辞职后一个人去了大理，在洱海边哭了一场"、"带70岁的奶奶第一次坐飞机看大海" | 情感共振驱动点赞和评论分享 |
| 保姆级攻略型 | 启思·价值 | "成都4天3晚保姆级攻略，人均1500全搞定"、"第一次去日本自由行完整攻略" | 超高实用价值，收藏率极高 |
| 反差对比型 | 破局·创意 | "花5000和花500住同一个城市，体验差别有多大"、"跟着外国博主的攻略玩中国" | 新奇对比视角驱动好奇心 |
| 避坑预警型 | 明道·洞见 | "去云南旅游最容易踩的5个坑"、"千万别在旺季去这些景点" | 损失规避心理，救急型内容 |
| 宝藏目的地型 | 破局·创意 | "中国的'小瑞士'居然在贵州"、"这个免费古镇比乌镇美10倍还没人" | 小众发现的新鲜感和优越感 |
| 省钱攻略型 | 启思·价值 | "穷游西藏15天只花了3000"、"免门票的10个5A景区" | 实用省钱价值直击钱包痛点 |
| 季节限定型 | 动心·共鸣 | "三月去婺源看油菜花完整攻略"、"冬天一定要去一次哈尔滨" | 时令紧迫感+美好憧憬驱动行动 |

### 3.2 选题叠加公式

```
爆款选题 = 目的地 × 人群场景 × 差异化视角 × 道心四法
```

- **热门目的地**：云南、成都、西藏、新疆、日本、东南亚、新晋网红城市
- **人群场景**：独旅、情侣、亲子、闺蜜、毕业旅行、银发旅行
- **差异化视角**：本地人视角、省钱极限、反季节、深度人文

### 3.3 周期性爆款节奏

- **春季（3-5月）**：赏花攻略、清明小长假、五一出行规划
- **夏季（6-8月）**：避暑目的地、毕业旅行、暑期亲子游
- **秋季（9-11月）**：赏秋红叶、十一长假攻略、错峰游推荐
- **冬季（12-2月）**：滑雪攻略、春节旅行、冬季温泉、跨年目的地

## 四、钩子模板

### 4.1 信息差型（明道·洞见）

```
"去{目的地}，{大多数人的做法}就亏了，{正确做法}才对。"
```
- "去西安只知道回民街就亏了，本地人都去这三条巷子。"
- "去泰国别只会去曼谷芭提雅，这个小城才是天花板。"
- "90%的人去厦门都走错了路线，按这个顺序玩效率翻倍。"

### 4.2 数据冲击型（启思·价值）

```
"{目的地}{天数}天，人均只要{低价格}，{高品质体验}全包。"
```
- "重庆5天4晚，人均800，火锅洪崖洞轻轨穿楼全安排。"
- "日本东京7天自由行，人均6000，含往返机票。"
- "西藏自驾15天，两个人总共花了8000，住得还不错。"

### 4.3 情感触发型（动心·共鸣）

```
"如果你最近{负面状态}，就去{目的地}走走吧。"
```
- "如果你最近很累很迷茫，就去大理住几天吧。"
- "工作三年没请过假，终于带妈妈去了她念叨了一辈子的北京。"
- "一个人的旅行，是给自己最好的礼物。"

### 4.4 悬念猎奇型（破局·创意）

```
"中国居然有个{意想不到的描述}的地方，99%的人没听过。"
```
- "中国居然有个'水上公路'，开车像在海上飞。"
- "藏在贵州深山里的瀑布，比黄果树还震撼，还免门票。"
- "这个南方小城居然下雪了，美得像走进了动漫。"

### 4.5 避坑警告型（明道·洞见）

```
"去{目的地}之前一定要知道这{N}件事，否则{负面后果}。"
```
- "第一次去云南，这5个坑千万别踩，血泪教训。"
- "暑假带娃去三亚，这些套路你必须提前知道。"
- "去日本自由行前必看，这些规矩不懂会很尴尬。"

### 4.6 限时紧迫型（动心·共鸣）

```
"再不去{目的地}就来不及了，{限时理由}。"
```
- "三月再不去婺源，油菜花就谢了，附完整攻略。"
- "这个古镇马上要收门票了，现在去还免费。"
- "今年冬天哈尔滨的冰雪大世界，错过再等一年。"

## 五、内容角度

### 5.1 保姆级攻略角度

**公式**：行程规划 + 交通住宿 + 费用明细 + 避坑提醒

```
选题示例："成都4天3晚保姆级攻略，照着走就行"
结构：行程总览 → Day1-4详细安排 → 交通方式 → 住宿推荐 → 必吃美食 → 费用汇总 → 避坑清单
```

**优势**：收藏率极高，长尾流量好，适合图文笔记

### 5.2 本地人视角

**公式**：本地身份背书 + 隐藏玩法 + 独家推荐

```
选题示例："在成都生活了8年，这些地方游客根本不知道"
结构：本地人身份引入 → 3-5个隐藏宝藏点 → 独家体验 → 为什么本地人喜欢去
```

**优势**：信息差价值高，可信度强，差异化明显

### 5.3 沉浸式旅行叙事角度

**公式**：故事线 + 情感变化 + 视觉美学 + 人文感悟

```
选题示例："一个人在拉萨待了七天，我想明白了很多事"
结构：出发动机 → 旅途中的人和事 → 感悟转变 → 金句收尾
```

**优势**：情感共鸣强，完播率高，评论区互动活跃

### 5.4 对比测评角度

**公式**：同类目的地/方案对比 + 优劣分析 + 推荐结论

```
选题示例："大理vs丽江，第一次去云南到底选哪个"
结构：两地概况 → 风景对比 → 消费对比 → 适合人群 → 最终建议
```

**优势**：解决决策焦虑，信息密度高

## 六、常见错误

### 6.1 内容层面

| 错误 | 具体表现 | 正确做法 |
|------|---------|---------|
| 滤镜过重 | 照片修到失真，实地落差大 | 适度调色，保持真实感，必要时标注"已调色" |
| 攻略过时 | 去年的价格/路线今年已变 | 标注信息时效，如"2026年3月实测" |
| 信息不全 | 只说好看不说怎么去 | 必须包含交通、费用、开放时间等实用信息 |
| 千篇一律 | 和其他攻略高度雷同 | 加入个人体验和独特视角，找差异化切口 |
| 只种草不避坑 | 全是美化没有真实提醒 | 优缺点都要说，增强可信度 |
| 地理常识错误 | 把昆明写成"离大理很近" | 核实距离、车程等基础信息 |

### 6.2 制作层面

- **画面抖动**：旅行vlog稳定器必备，手持抖动影响观感
- **解说太少**：纯风景没有解说信息量低，加入旁白和字幕
- **封面同质化**：避免千篇一律的风景照，加文字标题提升点击率
- **时长过长**：攻略类建议60-90秒精华版+评论区详版，叙事类不超过3分钟

## 七、合规红线

### 7.1 旅游宣传红线

- **禁止**虚假宣传目的地（如严重滤镜失真、编造不存在的景点）
- **禁止**鼓励违规行为（如闯入未开放区域、踩踏文物、投喂野生动物）
- **禁止**发布可能引发安全事故的极限打卡内容（如悬崖边拍照）
- **禁止**未标注广告的推广内容（酒店/景区合作必须标注"广告"或"合作"）

### 7.2 文化与政治红线

- **禁止**对少数民族文化的曲解或不尊重表达
- **禁止**涉及边境敏感地区的未经审批内容
- **禁止**歪曲历史遗迹、革命纪念地的文化内涵
- **谨慎**涉及宗教场所内容，尊重宗教信仰和习俗

### 7.3 AI内容合规

- AI生成的旅行攻略必须人工核实信息准确性（地址、价格、开放时间）
- AI生成的风景图片/视频必须标注
- 禁止用AI合成虚假旅行经历冒充真实体验

### 7.4 安全表达替换表

| 敏感表达 | 安全替代 |
|---------|---------|
| "此生必去" | "非常值得一去" |
| "全网最全攻略" | "非常详细的攻略分享" |
| "最便宜的玩法" | "性价比很高的玩法" |
| "绝对不会踩雷" | "踩雷概率很低" |
| "只有本地人知道" | "相对小众的宝藏地" |

## 八、AI Prompt 注入要点

```
【旅行攻略赛道知识注入·道心四法】
赛道特性：决策驱动+视觉驱动+情感驱动，受众22-40岁为主，兴趣出游是2026核心趋势。
道心四法映射：
- 明道·洞见：信息差类（本地人视角/避坑揭秘/冷门宝藏），制造"原来应该这样玩"的认知落差
- 动心·共鸣：叙事类（旅途感悟/家人旅行/治愈独旅），触发向往/感动/共鸣情绪
- 启思·价值：攻略类（保姆级路线/省钱方案/行程规划），高收藏率，强调具体时间+费用+交通
- 破局·创意：猎奇类（反差对比/隐秘目的地/特殊玩法），新鲜感驱动传播
钩子规则：前3秒出现震撼风景或冲突信息，触发词：免费、冷门、本地人、避坑、人均X元。
禁区：不虚假宣传，不过度滤镜，攻略信息必须标注时效，AI内容需标注，合作内容标注广告。
结尾公式：实用信息汇总+互动（"你们最想去哪里？评论区聊聊"）。
```
