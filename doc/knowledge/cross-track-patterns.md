# 跨赛道通用模式 知识库

> 适用于「道心文案」AI Prompt 注入，提供国学/道家养生/修心三大赛道的交叉规律与通用策略。

---

## 一、三赛道共性分析

### 1.1 共享受众画像

三个赛道的用户存在高度重叠，核心共享人群为：

| 共享人群 | 人群特征 | 在三赛道中的表现 |
|---------|---------|----------------|
| 30-45岁职场中产 | 有一定文化素养，面临人生中段困惑 | 国学找智慧、养生保健康、修心求平静 |
| 35-55岁女性 | 家庭角色繁重，关注身心健康 | 养生是入口 → 修心是升级 → 国学是深化 |
| 文化自信青年（25-35） | 对传统文化有兴趣和认同 | 国学是兴趣 → 修心是需求 → 养生是补充 |

**关键发现**：一个用户往往同时关注2-3个赛道，用户在赛道间的流转路径通常是：
```
养生（身体入口）→ 修心（情绪升级）→ 国学（认知深化）
```
或反向：
```
国学（认知入口）→ 修心（内化）→ 养生（实践）
```

### 1.2 共享内容基因

| 共性维度 | 具体表现 |
|---------|---------|
| 智慧源头 | 三赛道共享同一批经典：《道德经》《黄帝内经》《论语》《庄子》《菜根谭》 |
| 表达调性 | 均偏向温和、深沉、有质感的表达，排斥浮躁和肤浅 |
| 价值观 | 崇尚内在修养、天人合一、知行合一、顺其自然 |
| 审美偏好 | 中式美学（水墨、书法、古风、留白、自然） |
| 用户需求 | 在快节奏时代寻找"慢下来的力量"和"不变的智慧" |
| BGM选择 | 古琴、箫、自然音、空灵音乐 |

### 1.3 共享情绪触发机制

三个赛道都依赖以下核心情绪触发：

```
1. 恍然大悟感 — "原来如此！古人早就说透了"
2. 被理解感 — "这说的就是我"
3. 文化自豪感 — "我们的老祖宗真有智慧"
4. 安心感 — "原来这样做就对了"
5. 醍醐灌顶感 — "看完突然想通了"
```

## 二、跨赛道内容策略

### 2.1 赛道融合选题矩阵

| 融合方向 | 选题示例 | 主赛道 | 辅赛道 |
|---------|---------|--------|--------|
| 国学 × 养生 | "《黄帝内经》里的四季养生总纲" | 养生 | 国学 |
| 国学 × 修心 | "王阳明心学：停止内耗的终极方法" | 修心 | 国学 |
| 养生 × 修心 | "情志养生：为什么生气真的会伤肝" | 养生 | 修心 |

| 三合一 | "庄子的人生哲学：养身先养心" | 修心 | 国学+养生 |

### 2.2 跨赛道引流路径

```
场景A：养生用户 → 修心内容
钩子："失眠不只是身体的问题，你的心也需要'休息'。"
路径：身体症状 → 情志原因分析 → 修心建议

场景B：国学用户 → 养生内容
钩子："《道德经》里藏着一套完整的养生体系，你知道吗？"
路径：经典引用 → 养生智慧提取 → 实操方法

场景C：修心用户 → 国学内容
钩子："你所有的焦虑，庄子在两千年前就给出了答案。"
路径：现代痛点 → 古典智慧 → 认知升级
```

### 2.3 IP人设融合建议

三赛道融合的理想人设形象：

```
核心人设：一位通透的传统文化生活家
- 懂国学但不掉书袋（学识渊博但说人话）
- 懂养生但不迷信（科学理性但尊重传统）
- 懂修心但不鸡汤（真诚深刻但不空洞）
- 最终呈现：一个活得明白、活得健康、活得通透的人
```

## 四、通用合规框架

### 4.1 三赛道共同红线

```
1. 禁止封建迷信内容（算命、风水、神鬼等）
2. 禁止绝对化表述（最、第一、唯一、必定）
3. 禁止虚假历史（伪造古人言论、篡改经典）
4. 禁止医疗效果宣称（养生赛道尤其严格）
5. 禁止心理诊断/治疗建议（修心赛道注意）
6. 禁止宗教传教内容（引用智慧 ≠ 传教）
7. 禁止贩卖焦虑后引流到付费产品
8. 注意AI内容标注要求
```

### 4.2 安全表达通用原则

```
用"传统认为/古人说/一种观点" 替代 "事实是/一定/绝对"
用"有助于/或许能/可以试试" 替代 "可以治疗/一定有效/保证"
用"个人感悟/分享" 替代 "教你/你应该/你必须"
用"值得参考" 替代 "唯一正确"
```

## 五、通用钩子公式库

### 5.1 适用三赛道的万能钩子结构

| 钩子类型 | 公式 | 适用赛道 |
|---------|------|---------|
| 反常识 | "你以为{常识}？其实{真相}。" | 国学、养生 |
| 情感共鸣 | "如果你{状态}，这段话说给你听。" | 修心、国学 |
| 身体信号 | "经常{症状}的人，注意了。" | 养生 |
| 经典引用 | "《{经典}》说：'{原文}'——{解读}。" | 国学、养生 |
| 时令紧迫 | "{时间节点}之后，{紧迫事项}。" | 养生、国学 |
| 人物故事 | "{人物}在{处境}中，说了{N}个字。" | 国学、修心 |
| 金句顿悟 | "一句话，点醒了我{N}年的{困惑}。" | 修心、国学 |
| 数字具象 | "每天{动作}{时间}，{周期}后{变化}。" | 养生、修心 |

### 5.2 跨赛道万能结尾模板

```
互动引导型：
- "你觉得呢？评论区聊聊。"
- "你是哪种体质/哪种感受？评论区告诉我。"
- "如果说到了你心里，点个赞让我知道。"

治愈收尾型：
- "愿你{美好祝愿}。"
- "记住：{简短金句}。"
- "送给每一个{目标人群}。"

悬念延续型：
- "下一期我们聊{预告话题}，关注我不迷路。"
- "这只是第一层，真正的智慧在下一条。"
```

## 六、AI Prompt 通用注入模板

以下为三赛道通用的系统提示补充，在具体赛道知识注入之前加载：

```
【传统文化内容通用规范】
1. 文化态度：尊重传统但不迷信，传承精华但承认时代局限
2. 表达风格：口语化、有温度、有节奏感、像朋友分享而非老师授课
3. 信息密度：3秒一个信息点或情绪变化，每15秒一个高光时刻
4. 引用规范：经典原文必须标注出处，人物故事不得虚构核心事实
5. 合规底线：不用绝对化表述、不做医疗建议、不涉封建迷信、不传教
6. 数据预埋：自然植入引发点赞的高见感、引发评论的开放问题、引发分享的社交货币
7. 视觉适配：文案要考虑配合水墨/古风/中式美学的画面
8. 声音适配：文案朗读出来要顺口，适合中等偏慢的语速
```
