## 四、钩子模板

### 4.1 认知反转型（明道·洞见）

```
"你一直觉得自己{负面认知}，其实不是{表面原因}，而是{深层原因}。"
```
- "你一直觉得自己穿什么都土，其实不是衣服的问题，是配色出了错。"
- "为什么你穿白T恤显胖？不是你胖，是你选错了版型。"
- "穿衣显矮不是因为个子小，是腰线位置不对。"

### 4.2 效果展示型（破局·创意）

```
"同一个人，换了{穿搭方式}之后，效果差距也太大了。"
```
- "同一件基础白衬衫，5种搭法从上班到约会全搞定。"
- "换个扎头发的方式，穿搭氛围感直接拉满。"
- "改造前后对比，同事都问我是不是瘦了10斤。"

### 4.3 身材共情型（动心·共鸣）

```
"{身材特征}的女生/男生，这样穿直接开挂。"
```
- "梨形身材的姐妹看过来，这样穿下半身直接消失。"
- "155小个子的春夏穿搭，每一套都显高10cm。"
- "微胖女孩别再穿oversize了，这种版型才真显瘦。"

### 4.4 场景救急型（启思·价值）

```
"明天要{场景}不知道穿什么？这{N}套直接抄。"
```
- "明天面试不知道穿什么？这3套不出错还加分。"
- "第一次见男朋友家长穿什么？照着这样穿准没错。"
- "年会穿搭指南，不踩雷还能成为全场焦点。"

### 4.5 省钱种草型（启思·价值）

```
"全身不超过{低价格}，穿出{高级效果}。"
```
- "全身穿搭不超过200块，同事以为我买了大牌。"
- "优衣库这5件基础款，搭出一整个秋冬的高级感。"
- "学生党必看，百元穿搭也能穿出千元质感。"

### 4.6 趋势预告型（明道·洞见）

```
"2026年{季节}最火的穿搭趋势来了，{核心信息}。"
```
- "2026春夏最火的5个色系，穿上直接走在潮流前端。"
- "今年秋冬'静奢风'怎么穿才不像卖保险的？关键在这三个细节。"
- "新中式穿搭火了一整年，但90%的人穿错了。"

## 五、内容角度

### 5.1 身材解决方案角度

**公式**：身材特征分析 + 穿搭原理 + 具体单品推荐 + 效果对比

```
选题示例："梨形身材穿搭全攻略，从选款到搭配一篇搞定"
结构：身材特征说明 → 穿搭核心原则（遮/露/比例） → 适合的单品类型 → 3-5套完整搭配展示
```

**优势**：精准命中目标人群，互动率高，适合系列化

### 5.2 一衣多穿角度

**公式**：一件单品 + 多种搭配方式 + 不同场景展示

```
选题示例："一件白衬衫的7种穿法，从周一穿到周日"
结构：单品展示 → 逐一搭配演示 → 场景说明 → 穿搭技巧tips
```

**优势**：实用性强，收藏率高，衬托单品价值

### 5.3 穿搭改造角度

**公式**：改造前形象 + 问题分析 + 改造方案 + 改造后效果

```
选题示例："帮路人素人改造穿搭，效果堪比整容"
结构：改造前全身拍 → 问题诊断 → 搭配方案 → 换装过程 → 前后对比
```

**优势**：视觉反差极强，完播率高，互动活跃

### 5.4 趋势解读角度

**公式**：流行趋势 + 适合人群 + 具体穿法 + 踩雷预警

```
选题示例："2026春夏'薄荷曼波'风到底怎么穿"
结构：趋势来源 → 核心元素解析 → 适合/不适合的身材 → 搭配示范 → 常见误区
```

**优势**：时效性强，搜索流量好，体现专业度

## 六、常见错误

### 6.1 内容层面

| 错误 | 具体表现 | 正确做法 |
|------|---------|---------|
| 脱离受众 | 推荐万元搭配给学生粉丝 | 了解受众消费能力，标注价格区间 |
| 只看不学 | 好看但不说尺码/品牌/链接 | 每套搭配标注品牌、尺码、大致价格 |
| 身材误导 | 85斤博主教"微胖穿搭" | 人设要真实，用真实身材展示才有说服力 |
| 忽略实穿性 | 摆拍好看但日常穿不出门 | 强调日常实穿场景，减少"仅供拍照"的搭配 |
| 配色灾难 | 全身颜色超过3种以上还不协调 | 教受众配色原则：同色系/对比色/万能色 |
| 没有原理 | 只说"好看"不说"为什么好看" | 讲清穿搭原理（腰线、比例、色彩），授人以渔 |

### 6.2 制作层面

- **光线不一致**：对比图需同一光线环境，否则效果不真实
- **镜头畸变**：广角拍全身会显矮显胖，用标准焦段或站远拍
- **背景杂乱**：穿搭展示需干净背景，突出衣服本身
- **节奏太快**：每套搭配至少停留3秒让观众看清全貌

## 七、合规红线

### 7.1 身材与容貌焦虑红线

- **禁止**制造身材焦虑（"胯宽的人穿什么都丑""腿粗就别穿裙子了"）
- **禁止**过度修图/滤镜修改身材比例（拉腿、缩腰等明显P图）
- **禁止**宣扬畸形审美（过度追求"白幼瘦""BM风"等不健康标准）
- **谨慎**使用"胖""矮""丑"等词汇，改用中性描述

### 7.2 商业合规红线

- **禁止**未标注的广告推广（品牌合作必须标注"广告"或"合作"）
- **禁止**虚假平替宣传（称仿冒品为"平替"）
- **禁止**使用"最时尚""第一名""全网最低价"等绝对化用语
- **禁止**以"打假""测评"名义进行不当获利或散布虚假信息

### 7.3 知识产权红线

- **禁止**直接使用品牌LOGO做封面（可能构成商标侵权）
- **谨慎**使用奢侈品牌名称进行对比（避免"高仿""1:1复刻"等表述）
- 引用他人穿搭图片需注明出处

### 7.4 AI内容合规

- AI生成的穿搭效果图/虚拟试穿必须标注
- 禁止用AI换脸冒充真人穿搭展示
- AI生成的时尚趋势预测需标注"AI分析，仅供参考"

### 7.5 安全表达替换表

| 敏感表达 | 安全替代 |
|---------|---------|
| "胖的人别穿这个" | "这个版型对圆润身材不太友好" |
| "腿粗的千万别穿" | "这款更适合腿部线条纤细的身材" |
| "最显瘦的穿法" | "视觉上非常显瘦的穿法" |
| "高仿大牌平替" | "质感类似的平价选择" |
| "穿了绝对被夸" | "穿了经常收到夸奖" |

## 八、AI Prompt 注入要点

```
【穿搭时尚赛道知识注入·道心四法】
赛道特性：视觉驱动+身份表达+消费决策，受众18-35岁女性为主，2026趋势：妆造一体化、圈层化、场景化。
道心四法映射：
- 明道·洞见：原理类（穿搭显瘦原理/配色法则/身材分析），制造"原来穿错了"的认知落差
- 动心·共鸣：身材类（梨形/小个子/微胖/产后穿搭），触发"这说的就是我"的共鸣+被理解感
- 启思·价值：公式类（万能搭配/场景穿搭/胶囊衣橱），高收藏率，强调可复制+标注品牌价格
- 破局·创意：改造类（素人改造/一衣多穿/混搭挑战），前后对比驱动传播
钩子规则：前3秒展示穿搭效果或身材痛点，触发词：显瘦、显高、高级感、一衣多穿、闭眼入。
禁区：不制造身材焦虑，不过度修图，不用绝对化用语，合作推广必须标注广告，禁用"高仿"表述。
结尾公式：完整搭配展示+互动（"你是什么身材？评论区告诉我帮你搭"）。
```
