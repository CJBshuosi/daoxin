# 考试考证赛道 垂直知识库

> 适用于「道心文案」AI Prompt 注入，提供考试考证赛道（考公、考研、职业资格证）的选题、钩子、合规红线等垂直知识。

---

## 一、赛道概况

- 2026年国考283.1万人参加笔试，录取比约74:1，考公热度持续高企
- 考研报名人数虽有回调但仍维持在400万+量级，"逆向考研""在职考研"成新趋势
- 职业资格证赛道持续扩容：法考、CPA、教资、一建二建、软考、PMP等均为高热赛道
- 抖音教育赛道日均播放量数百亿，考试类内容属"正能量"品类，平台持续扶持
- 小红书偏重"上岸经验帖+备考方法论"图文，抖音偏重"名师IP讲解+技巧速记"短视频
- 核心变现路径：在线课程（客单价99-9999元）、资料包引流私域、训练营、图书带货
- 2025-2026年热点方向：AI辅助备考工具、考公年龄放宽（上限38岁）、在职备考、跨专业考研
- 行业已从流量争夺期进入精细化运营期，拼内容质量和教学效率

## 二、受众画像

### 2.1 核心人群

| 人群标签 | 占比（估算） | 特征 |
|---------|------------|------|
| 22-28岁应届/往届毕业生 | 35% | 考公考研刚需，时间相对充裕，信息焦虑严重 |
| 25-35岁在职备考人群 | 25% | 时间碎片化，追求高效方法，愿意付费 |
| 20-25岁大学在校生 | 20% | 考研/考证预备阶段，偏好免费资料和经验分享 |
| 30-38岁二次考公人群 | 10% | 年龄放宽政策利好群体，焦虑感强，决策果断 |
| 备考家属（父母/伴侣） | 10% | 关注备考心理调适、就业前景等周边内容 |

### 2.2 用户痛点与需求

- **方法焦虑**：备考方法太多不知道选哪个，反复切换效率低
- **时间不够**：在职备考时间碎片化，不知如何合理分配
- **坚持困难**：备考周期长（6-18个月），中途容易放弃
- **信息差焦虑**：不了解报考政策、岗位选择技巧、分数线变化
- **选岗迷茫**：不知道哪些岗位竞争小、上岸概率高
- **心态崩溃**：越临近考试越焦虑，模考成绩波动大
- **二战压力**：一战失败后的心理重建和方法调整

```
上岸、逆袭、高效、速记、真题、押题、必考、
零基础、在职备考、一战上岸、时间规划、背诵技巧、
```

## 三、爆款选题类型（映射道心四法）

### 3.1 选题类型矩阵

| 选题类型 | 典型选题 | 道心四法 | 爆款原因 |
|---------|---------|---------|---------|
| 认知纠偏型 | "考公不是死记硬背——行测80+的人都用这个思维方式"、"考研政治不要从7月才开始" | 明道·洞见 | 打破备考误区，认知落差感强 |
| 信息差揭秘型 | "这5个公务员岗位竞争比低到离谱"、"考研调剂的隐藏规则，大多数人不知道" | 明道·洞见 | 独家信息感，高收藏高分享 |
| 上岸故事型 | "从三本到985研究生，我用了两年"、"32岁在职妈妈一战上岸省考" | 动心·共鸣 | 代入感强，激励效果显著 |
| 备考日常型 | "在职考公人的一天""考研倒计时30天的凌晨" | 动心·共鸣 | 陪伴感强，评论区抱团取暖 |
| 方法清单型 | "行测资料分析速算6大公式"、"考研英语阅读四步解题法" | 启思·价值 | 拿来即用，收藏率极高 |
| 规划模板型 | "零基础6个月考公上岸时间表"、"考研全年复习规划（精确到周）" | 启思·价值 | 系统性强，长尾流量好 |
| 反常识观点型 | "别考公务员——除非你想清楚了这三件事"、"考研不一定要辞职，在职反而更高效" | 破局·创意 | 逆向思维引发讨论和争议互动 |
| 跨界类比型 | "用游戏思维备考行测""把考研当成一次创业来管理" | 破局·创意 | 新奇视角降低备考枯燥感 |

### 3.2 选题公式

```
爆款选题 = 考试类型/阶段痛点 × 方法/信息差 × 可执行性/情绪共鸣
```

- **考试类型**：考公（国考/省考）、考研、法考、CPA、教资、一建二建
- **阶段痛点**：择校选岗、基础阶段、强化冲刺、考前心态、查分调剂
- **可执行性**：时间规划、背诵技巧、解题模板、资料推荐

### 3.3 周期性爆款节奏

| 时间节点 | 爆款方向 |
|---------|---------|
| 1-3月 | 考研复试/调剂、省考报名、新一年备考规划 |
| 4-6月 | 省考笔试、考研择校、法考/CPA备考启动 |
| 7-9月 | 考研强化阶段、国考公告、教资笔试 |
| 10-12月 | 国考笔试、考研冲刺、年度总结与来年规划 |

## 四、钩子模板

### 4.1 误区纠正型（明道·洞见）

```
"如果你{错误的备考方式}，那你可能在浪费时间。"
```
- "如果你还在一道一道刷行测真题，那你已经落后了。80+的人都在用模块突破法。"
- "考研英语背单词从A开始背？难怪你总是背不完。正确的顺序应该是这样。"
- "90%的人复习申论都在抄范文，这恰恰是申论上不了70的原因。"

### 4.2 信息差型（明道·洞见）

```
"关于{考试/岗位}，有一个{大多数人不知道的信息}。"
```
- "考公年龄已经放宽到38岁了，但这三类岗位的隐性年龄门槛你必须知道。"
- "考研有一种'捡漏'方式叫专项计划，很多人根本不知道自己符合条件。"
- "这5个证书含金量被严重低估，考过直接涨薪30%以上。"

### 4.3 上岸经验型（动心·共鸣）

```
"从{起点}到{终点}，我做对了{关键事情}。"
```
- "从行测55到78，我只做了一个改变：不再盲目刷题，而是先吃透这套方法论。"
- "三本出身、二战上岸985，我整理了走过的所有弯路，希望你不要再走一遍。"
- "在职备考6个月一战上岸省考，我的时间分配表分享给你。"

### 4.4 痛点直击型（动心·共鸣）

```
"如果你{备考困境}，问题大概率出在{根本原因}。"
```
- "如果你行测做不完题，不是手速不够快，是你的做题顺序有问题。"
- "如果你考研政治选择题总在30分徘徊，不是背得不够多，是逻辑框架没建起来。"
- "如果你每天学10个小时还觉得没进步，你需要的不是更多时间，而是复盘。"

### 4.5 高效清单型（启思·价值）

```
"{N}个{考试科目}的{核心技巧/资料}，{效果承诺}。"
```
- "申论大作文5种万能开头模板，背熟了至少多拿5分。"
- "考研数学高频考点30条，考前一周反复看。"
- "法考主观题必背的8个答题框架，直接套用不丢分。"

### 4.6 规划指导型（启思·价值）

```
"距离{考试}还有{时间}，按这个节奏复习来得及。"
```
- "距离国考还有3个月，零基础按这个计划走，完全来得及。"
- "考研最后60天，每天这样安排，效率翻倍。"
- "在职备考CPA，每天只有2小时？这个方案专为你设计。"

## 五、内容角度

### 5.1 方法拆解角度（高收藏，核心内容）

**公式**：科目痛点 + 方法介绍 + 步骤拆解 + 实战示范

```
选题示例："行测资料分析如何3分钟做完一组题"
结构：痛点引入（做不完题）→ 方法概述（估算法）→ 步骤演示 → 练习建议
```

### 5.2 信息差角度（高分享+涨粉）

**公式**：冷门信息 + 价值说明 + 操作指南 + 注意事项

```
选题示例："这些'神仙岗位'只在省考里有，国考根本没有"
结构：信息抛出 → 岗位分析 → 报考条件 → 备考建议
```

### 5.3 经验复盘角度（高互动+信任建立）

**公式**：个人背景 + 备考历程 + 关键转折 + 方法总结

```
选题示例："我如何从行测55分提到80分——一个笨人的逆袭方法"
结构：初始水平 → 走过的弯路 → 转折点和方法调整 → 最终结果 → 建议
```

### 5.4 心态调适角度（高点赞+评论）

**公式**：焦虑场景 + 共情理解 + 心理调适 + 行动建议

```
选题示例："考前一周焦虑到睡不着？这很正常，试试这三个方法"
结构：描述焦虑状态 → 说明这是正常反应 → 具体调适方法 → 鼓励
```

## 六、常见错误

### 6.1 内容层面

| 错误 | 具体表现 | 正确做法 |
|------|---------|---------|
| 虚假承诺 | "保证上岸""必过四级""押中原题" | 合理设置预期，"掌握方法能显著提高效率" |
| 贩卖焦虑 | "今年不上岸这辈子就完了""35岁还没考公就晚了" | 提供信息和方法，而非制造恐慌 |
| 信息过时 | 用去年的政策、分数线、考纲讲解 | 确保政策和数据是最新版本 |
| 内容同质化 | 反复搬运"早起打卡""鸡汤激励" | 提供信息增量，差异化知识点或独特视角 |
| 一刀切建议 | 不区分在职/全职、一战/二战、基础好坏 | 明确标注适用人群和前提条件 |
| 纸上谈兵 | 没有实战经验却教备考方法 | 分享有据可查的经验或引用权威来源 |

### 6.2 制作层面

- **知识点密度过高**：一条视频塞太多内容，用户记不住；单条视频聚焦1-2个核心知识点
- **缺少可视化**：纯口播讲方法论效果差，配合思维导图、时间表等图表更直观
- **节奏拖沓**：备考类用户时间宝贵，开头5秒内必须切入主题
- **字幕缺失**：关键数据、公式、步骤必须有文字强调

## 七、合规红线

### 7.1 教育培训广告合规

- **禁止**暗示有命题专家参与培训（"命题组老师亲授""内部押题"）
- **禁止**做出保证性承诺（"保过""保上岸""不过全额退"需有实际退费机制）
- **禁止**宣称"圈定考试范围""考前密押原题"
- **禁止**冒充官方机构或利用官方名义宣传
- 涉及具体考试成绩提升的需标注"效果因人而异"

### 7.2 极限用语合规

- **禁止**使用"最好的备考方法""唯一的上岸路径""第一名师"等绝对化用语
- 安全替代："更高效的方法""经过验证的备考路径""资深讲师"
- 时限性表述须有具体时限（"限时三天"需标明起止日期），禁止"随时涨价""仅此一次"

### 7.3 知识产权合规

- **禁止**泄露真题或暗示押中原题（考试真题版权归命题机构所有）
- **禁止**未经授权搬运他人课程内容
- 使用教材内容须注意版权，不得完整展示
- 引用公开数据须标注来源

### 7.4 平台规则

- 抖音/小红书对教育培训类内容有专项审核机制
- 评论区不得直接放课程链接/联系方式（引流需通过合规组件）
- AI生成的教学内容建议标注
- 面向未成年人的内容需特别注意合规

### 7.5 安全表达替换表

| 敏感表达 | 安全替代 |
|---------|---------|
| "保证上岸/保过" | "帮助你系统提升，大幅提高上岸概率" |
| "押中原题/内部资料" | "基于历年考情分析的高频考点梳理" |
| "最好的老师/机构" | "经验丰富的讲师/口碑不错的机构" |
| "不学就来不及了" | "早准备多一分从容，现在开始刚刚好" |
| "7天速成/一周搞定" | "7天集中突破核心考点，打好基础框架" |
| "别人都在学你还不学" | "提前规划备考节奏，效率更高" |

## 八、AI Prompt 注入要点

```
【考试考证赛道知识注入】
1. 核心价值：提供高效备考方法+信息差，让用户觉得"方向对了，努力才有意义"
2. 钩子优先级：误区纠正 > 信息差揭秘 > 高效方法清单 > 上岸经验
3. 道心四法映射：备考误区/信息差=明道，上岸故事/备考日常=动心，方法清单/规划模板=启思，反常识观点=破局
4. 内容须标注适用考试类型（考公/考研/法考等）和适用人群（零基础/在职/二战）
5. 禁止：保证上岸、押题暗示、泄露真题、贩卖焦虑、冒充官方
6. 语气：像一位上岸的学长/学姐在分享经验，专业、务实、有温度
7. 数据和政策须注明年份，避免过时信息误导
8. 情绪曲线：焦虑共鸣→误区点破→方法给到→信心鼓励
9. 结尾引导："需要完整资料的评论区留言""你在备考什么？评论区交流"
10. 触发情绪：恍然大悟感、方向感、被理解感、"原来还能这样"的惊喜感
```
